Gtk::Scaleによるスライドバー

概要

Gtk::HScaleまたはGtk::VScaleによってスライドバーを作成する。

基本的には SpinButtonと同様に

・Gtk::Scale::set_range() で範囲をセット
・Gtk::Scale::set_increments() で変化量をセット
・Gtk::Scale::set_value() で初期値をセット

するだけで良い。

ソース

scale.cpp 
#include <gtkmm.h>

class MainWin : public Gtk::Window
{
    Gtk::HScale m_hscale;

public:
    MainWin();
};


MainWin::MainWin()
{
    m_hscale.set_range( 1, 5 );
    m_hscale.set_increments( 1, 1 );
    m_hscale.set_value( 2 );

    add( m_hscale );
    show_all_children();
}

int main( int argc, char *argv[] )
{
    Gtk::Main kit( argc, argv );
    MainWin mainwin;
    Gtk::Main::run( mainwin );

    return 0;
}

コンパイル

必要なコンパイルオプションは pkg-config を使って取得する。 

g++ scale.cpp -o scale `pkg-config gtkmm-2.4 --cflags --libs`
