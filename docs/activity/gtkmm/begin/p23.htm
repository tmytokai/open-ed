ウィジットの背景色変更

概要

ウィジットの背景色を変更する。

背景色は Gtk::Widget::modify_bg() によって変更する。modify_bg()の第一引数はウィジットの状態であり、

STATE_NORMAL : 通常
STATE_ACTIVE : アクティブ
STATE_PRELIGHT : マウスがウィジットの上にある
STATE_SELECTED : ボタンなどの場合に選択した
STATE_INSENSITIVE : ウィジットの状態変更不可

の5つの状態がある。第二引数は Gdk::Color であり色を指定する。前景色なども同様に modify_fg() などで変更できる。

ソース

backc.cpp 

#include <gtkmm.h>

class MainWin : public Gtk::Window
{
public:
    MainWin();
};

MainWin::MainWin()
{
    // 黄色をセット
    Gdk::Color col;
    col.set_rgb( 65535, 65535, 0 );

    // 背景色変更
    modify_bg( Gtk::STATE_NORMAL, col );
}

int main( int argc, char *argv[] )
{
    Gtk::Main kit( argc, argv );
    MainWin mainwin;
    Gtk::Main::run( mainwin );

    return 0;
}

コンパイル

必要なコンパイルオプションは pkg-config を使って取得する。 

g++ backc.cpp -o backc `pkg-config gtkmm-2.4 --cflags --libs`

結果

起動すると黄色のウィンドウが開く
