なし

最終更新日: 2017/10/31

目的

あなたが git を使ってファイルのバージョン管理ができる技術者になること

必要性

git はソースコード等のファイルの変更履歴を適切かつ高速に管理するためのシステムです。
チーム開発を円滑に進めるため、git の使い方を技術者は学ぶ必要があります。

進め方

もし上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
またこのアクティビティの具体的な進め方についてはこちらを参照して下さい。

スケジュール

このアクティビィティのスケジュールは以下の通りです。
