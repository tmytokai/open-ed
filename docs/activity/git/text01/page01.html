  2. 学習項目: [1] Git とは

ソフトウェア開発の規模が大きくなるにつれ、「バージョン管理システム」を使ってソースコードやドキュメント等のファイルのバージョンを管理する必要性が出てきます。
また一般的にバージョン管理システムはクラウド上にあり、簡単にバージョンの一元管理ができますのでチーム開発も容易になります。

バージョン管理システムには無料のものから有料のものまで沢山存在します。
今回のアクティビティでは世界で最も良く使用されているバージョン管理システム「Git(ギット)」の演習を行います。

ちなみにGitの作者はLinuxの作者のリーナス・トーバルズです。
