  2. 学習項目: [1] git とは

git を使うことでユーザーはある時点におけるファイルの内容を「リポジトリ」と呼ばれるデータベースに保存できます。
この「リポジトリ」にアクセスすることでユーザーはあるファイルの内容を以前のものに復元したり、編集した箇所の差分を表示したりすることができます。

この「リポジトリ」には「ローカルリポジトリ」と「リモートリポジトリ」の2種類あって、それぞれ

  • ローカルリポジトリ: 開発者の手元のPC内にあり、その開発者がひとりで利用するためのプライベートなリポジトリ
  • リモートリポジトリ: クラウド上にあり、開発チームに所属する複数の開発者が共同利用するリポジトリ

という違いがあります。
