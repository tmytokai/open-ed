1. アクティビティ: git によるバージョン管理
  2. 学習項目: [3] リモートリポジトリ

演習時間: 60 分


1. リモートリポジトリのクローン


演習 3-1 (チーム、10分): チーム内でA、B、C、D 役を決め、各自指定されたアドレスからリモートリポジトリ「gitremote」をクローンして状態 1 にして下さい。


2. ローカルリポジトリでの作業


演習 3-2 (チーム、10分): 各自 hoge.txt に A〜D を入力し(改行忘れずに)、git add、git commit して状態 2 にして下さい。


3. 更新内容のプッシュ (衝突が無い場合)


演習 3-3 (チーム、10分): Aさんはテキストに従ってプル、プッシュを行なって 状態 3 にして下さい。


4. 更新内容のプッシュ (衝突が有る場合)


演習 3-4 (チーム、10分): Bさんはテキストに従ってプル、修正、コミット、プッシュを行なって 状態 5 にして下さい。


演習 3-5 (チーム、10分): Cさん、Dさんは順にテキストに従ってプル、修正、コミット、プッシュを行なって 状態 6 にして下さい。


5. 「その他」


演習 3-6 (チーム、10分): A、B、Cさんはプルを行なって 状態 7 にして下さい。
