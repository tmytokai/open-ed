1. アクティビティ: git によるバージョン管理
  2. 学習項目: [3] リモートリポジトリ

演習時間: 120 分


1. リモートリポジトリのクローン


演習 3-1 (チーム、20分):

チーム内でA、B、C、D 役を決めて下さい
A さんはテキストに従ってリモートリポジトリ「gitremote」を作成し、B、C、Dさんをメンバー登録して下さい。
A、B、C、D さんはリモートリポジトリをローカルに clone して状態 1 にして下さい。
A さんは「Collaborators」ページのスクリーンショットを指定した場所に貼り付けて下さい。


2. ローカルリポジトリでの作業


演習 3-2 (チーム、20分):

各メンバーはテキストに従って hoge.txt を更新し、add、commit して状態 2 にして下さい。
提出物はありません。


3. 更新内容のプッシュ (衝突が無い場合)


演習 3-3 (チーム、20分):

A さんはテキストに従って pull、push を行なって 状態 3 にして下さい。
提出物はありません。


4. 更新内容のプッシュ (衝突が有る場合)


演習 3-4 (チーム、20分):

Bさんはテキストに従って pull、修正、add、 commit、 push を行なって 状態 5 にして下さい。
提出物はありません。


演習 3-5 (チーム、20分):

Cさん、Dさんの順にテキストに従って pull、修正、add、 commit、 push を行なって 状態 6 にして下さい。
提出物はありません。


5. 「その他」


演習 3-6 (チーム、20分):

A、B、Cさんはテキストに従って pull を行なって 状態 7 にして下さい。
代表者はリモートリポジトリ「gitremote」のスクリーンショットを指定した場所に貼り付け、図の下に hoge.txt の最終的な内容をコピペして下さい。
