  2. 学習項目: [3] リモートリポジトリ

次は各開発者が自分のローカルリポジトリ内で自分の作業を行いますが、次の点を常に念頭に置いて作業すると良いでしょう。

「開発者間のローカルリポジトリは完全に独立しているので、自分のローカルリポジトリ内だけで作業している限り、何をしても他の開発者に影響は無い」

では以下の手順で各自が作業を行って状態2まで進めましょう。


hoge.txt をエディタで開いて name? の後で改行して自分の記号(A〜D)を入力します。
記号の後に更に１回改行して以下の状態にして下さい。なお 4 行目以降には空行が入らない気をつけて下さい。

name?(改行)
(A〜D)(改行)
(改行)(ファイル終端)

git add hoge.txt
git commit -m "my name is (A〜B)"

状態 2 の時点では他の人やリモートリポジトリに何の影響も与えていないことに注目して下さい。
