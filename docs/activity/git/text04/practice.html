  2. 学習項目: [4] GitHub Pages


1. 初期設定


演習 4-1 (個人): リポジトリを新規作成し、GitHub Pages の初期設定をしましょう。

リポジトリ名は「pages」とします
MDファイルを置くフォルダは「/docs」とします。
index.md は GitHub 上で直接編集します。
コミットしてブラウザで Web ページが表示されるか確認して下さい。
提出物はありません。


2. マークダウン(MD)ファイル


演習 4-2 (個人): 2ページ目に書いてある例を全て試してみましょう。

任意のフリー画像を探して hoge.png という名前で保存します。
hoge.png を docs フォルダにアップロードします。
例を全て試してみます。
コミットしてブラウザで Web ページが表示されるか確認して下さい。
指定場所に「Web ページのアドレス (※ リポジトリのアドレスではありません)」をコピーして下さい。
