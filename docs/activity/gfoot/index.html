目的

あなたが Greenfoot を使って初歩的なオブジェクト指向型開発について学ぶこと

必要性

現在主流となっているオブジェクト指向型のソフトウェア開発手法は、従来のウォーターフォール型の開発手法とそのやり方が大きく異なります。
ただし、初心者がいきなり本格的なオブジェクト指向型開発手法について学ぼうとしても難しすぎて挫折します。
従って本格的なオブジェクト指向型開発について学ぶ前に、Greenfoot を使った初歩的なオブジェクト指向型開発に技術者は慣れる必要があります。

進め方

もし上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
またこのアクティビティの具体的な進め方についてはこちらを参照して下さい。
