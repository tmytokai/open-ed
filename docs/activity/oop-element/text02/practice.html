仕様:

Score クラスに name、math、english の getter を追加する
Score クラスに name の setter を追加する
Taro のコンストラクタで name に "TARO" をセットする
Hanako のコンストラクタで name に "Hanako" をセットする
getter、setter を使う様に Main クラスの表示部を書き換える

結果:

引数なしコンストラクタ実行
name = TARO math = 28 english = 46 平均 = 37.0 合計 = 74
