3. getter と setter

演習 2-2 (個人、15分): 「setter」を使って、Taro と Hanako サブクラスのコンパイルエラーを消しましょう

演習 2-1 で作った「OELM-2-1」をフォルダごとコピーし、フォルダ名を「OELM-2-2」とします。
private なフィールドにはサブクラスからもアクセス出来ないので、name の値をセットする setter( 名前は public void set_name(String _name) ) を Score クラスに追加します。
Taro クラスのコンストラクタ Taro() の中で setter を使って name フィールドに "TARO" を代入して下さい。
同様に Hanako クラスのコンストラクタ Hanako() の中で setter を使って name フィールドに "HANAKO" を代入して下さい。
TaroクラスとHanakoクラスをコンパイルするとエラーメッセージが表示されないことを確認して下さい。
提出物はありません


演習 2-3 (個人、15分): 「getter」を使って全てのコンパイルエラーを消しましょう。

演習 2-2 で作った「OELM-2-2」をフォルダごとコピーし、フォルダ名を「OELM-2-3」とします。
name の値を取得する getter (名前は public String get_name() ) を Score クラスに追加します。
math の値を取得する getter (名前は public int get_math() ) を Score クラスに追加します。
english の値を取得する getter (名前は public int get_english() ) を Score クラスに追加します。
Main クラスの表示部を getter を使うように書き換えます。
