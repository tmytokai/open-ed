3. getter と setter


演習 2-2 (個人): 「setter」を使って、Taro と Hanako サブクラスのコンパイルエラーを消しましょう

フォルダ「OELM_2_1」をフォルダ「OELM_2_2」に丸ごとコピーし、その中に移動します。
Score.java を開きます。
private なフィールドにはサブクラスからもアクセス出来ないので、name の値をセットする setter( 名前は public void set_name(String _name) ) を Score クラスに追加します。
Taro.java を開き、Taro クラスのコンストラクタ Taro() の中で (3)で定義した setter を使って name フィールドに "TARO" を代入して下さい。
同様に Hanako.java を開き、 Hanako クラスのコンストラクタ Hanako() の中で setter を使って name フィールドに "HANAKO" を代入して下さい。
Taro.java と Hanako.java をコンパイルするとエラーメッセージが表示されなくなったことを確認して下さい。
提出物はありません
