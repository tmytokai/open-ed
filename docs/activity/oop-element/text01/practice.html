2. フィールドとメソッドの再利用

演習 1-2 (個人PC、15分): オブジェクト指向プログラミングの基礎の 演習で使った Score クラスをスーパークラスとし、フィールドとメソッドの再利用を試してみましょう。

新規のJavaプロジェクトを作ります。プロジェクト名は「OELM-1-2」とします。
Score クラスを作成し、「OBSC-1-14」の Score クラスの中身をコピーします。
Score スーパークラスを継承した Taro クラスを作成して下さい(中身は空のまま) 。
Score スーパークラスを継承した Hanako クラスを作成して下さい(中身は空のまま) 。
Main クラスを作成して下さい。
Taro クラスのインスタンス taro を作成して下さい(コンストラクタは引数無し)。
同様に Hanako クラスのインスタンス hanako を作成して下さい(コンストラクタは引数無し)。
taro と hanako の情報を以下のようにして表示して下さい。 

System.out.println( "太郎: math = " + taro.math+ " english = " + taro.english + " 平均 = " +taro.average());
System.out.println( "花子: math = " + hanako.math+ " english = " + hanako.english + " 平均 = " +hanako.average());

実行します。Taro と Hanako クラスの中身が空っぽなのにちゃんと動作していることを確認して下さい。
チーム内で答え合わせをし、提出用のソースコードを選んで下さい。
指定場所に「ソースコード」と「実行結果」を記入して下さい。
