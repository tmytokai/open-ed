2. フィールドとメソッドの再利用

演習 1-2 (個人、15分): オブジェクト指向プログラミングの基礎の 演習で使った Score クラスをスーパークラスとし、フィールドとメソッドの再利用を試してみましょう。

フォルダ「OELM-1-2」を新規作成し、その中に移動します。
「OBSC-1-14」の Score.javaをコピーします。
Taro.java を新規作成し、Score スーパークラスを継承した Taro クラスを定義して下さい(クラスの中身は空のまま) 。
Score.java を新規作成し、Score スーパークラスを継承した Hanako クラスを定義して下さい(クラスの中身は空のまま) 。
Main.java を新規作成し、以下のソースコードを貼り付けてください。

	public static void main(String[] args) {

		Taro taro = new Taro();
                Hanako hanako = new Hanako();

                System.out.println( "太郎: math = " + taro.math+ " english = " + taro.english + " 平均 = " +taro.average());
                System.out.println( "花子: math = " + hanako.math+ " english = " + hanako.english + " 平均 = " +hanako.average());
	}
}

実行します。Taro と Hanako クラスの中身が空っぽなのにちゃんと動作していることを確認して下さい。
