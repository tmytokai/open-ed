インヘリタンスとは


ある定義済みのクラスを親とし、親が持つ機能を受け継ぎつつ新しい機能を追加して子クラスを作り出すことを「インヘリタンス」または「継承」と言います。
この親クラスの事を「スーパークラス」や「基底クラス」「基本クラス」などと呼びます。また子クラスの事を「サブクラス」や「派生クラス」と呼びます。

例えばスーパークラスとして次の中身が空っぽの果物(FRUIT)クラスが定義されているとします。 

ソース1:

public class FRUIT {
}

次に果物(FRUIT)を継承したリンゴ(APPLE)サブクラスを定義してみます。 

ソース2:

public class APPLE extends FRUIT{

	public String name;
	
	public APPLE()
	{
		name = "りんご";
	}
	
	public void eat()
	{
		System.out.println(name+"美味い");
	}
}

ソース2のクラス名(APPLE)の後に出てきた「extends FRUIT」によりリンゴ(APPLE)が果物(FRUIT)を継承して作られたことを表しています。

同様に果物(FRUIT)を継承したみかん(ORANGE)サブクラスも定義してみましょう。 

ソース3:

public class ORANGE extends FRUIT{

	public String name;
	
	public ORANGE()
	{
		name = "みかん";
	}
	
	public void eat()
	{
		System.out.println(name+"美味い");
	}

}


さて継承によって定義したサブクラスは普通のクラスと同様にインスタンスを作って利用できます。
例えばソース4に示した main メソッドを実行した結果が実行結果1です。
コンストラクタで名前(name)に「りんご」「みかん」を代入してますのでインスタンスappleとorangeで表示が異なっています。 

ソース4:

public static void main(String[] args) {
	APPLE apple = new APPLE();
	ORANGE orange = new ORANGE();

	apple.eat();
	orange.eat();
}

実行結果1:

りんご美味い
みかん美味い
