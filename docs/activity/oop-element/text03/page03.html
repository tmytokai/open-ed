次に「メソッドによるポリモーフィズム」について簡単な例を示します。

これは「同じクラスの同じ名前のメソッドを使ってもインスタンス毎に動作が異なる」という性質のことで、やはり今まで既に演習で体験済みです。

例えばソース 1 の FRUIT クラスに対してソース 2 の main メソッドを実行してみます。

ソース 1 : 
public class FRUIT {

    public String name;

    public void eat(){
        System.out.println( name + "美味しい");
    }
}

ソース 2 : 
public class Main{

	public static void main(String[] args) {

	        FRUIT apple = new FRUIT();
	        FRUIT orange = new FRUIT();

	        apple.name = "りんご";
	        orange.name = "みかん";

	        apple.eat();
	        orange.eat();
	}
}

当然実行結果は次の様になります。

実行結果 1 : 
りんご美味しい
みかん美味しい

この様に、同じ FRUIT クラスの同じ eat() メソッドを呼び出しているのにインスタンス apple と orange で結果が異なっています。
