目的

あなたが WBS とカンバンを使ってスケジュール管理ができる技術者になること

必要性

WBS（Work Breakdown Structure）とカンバンはプロジェクトに含まれる作業工程を洗い出してスケジューリングを行うための技法です。
チーム開発を円滑に進めるため、WBS とカンバンの使い方を技術者は学ぶ必要があります。

進め方

上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
アクティビティの具体的な進め方についてはこちらを参照して下さい。
