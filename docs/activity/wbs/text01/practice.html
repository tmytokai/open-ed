4. WBS の作り方の例

演習 1-4 (個人PC、30分): ひとりで家にいる時にお腹が空いたので何か食べることにしました。
しかし手元には水くらいしかありません。いよいよお腹も空いてきました。
そこで WBS を作成して「ひとりで効率良く袋入りインスタントラーメンを作って食べる手順」を考えることにしました。
条件は以下のとおりです。

  • チームメンバー: 自分のみ
  • 袋入りインスタントラーメン: 無し
  • 水・鍋・ガス・コンロ・箸・どんぶり: 有り
  • その他食材・調味料: 一切無し

テキストを参考にし、任意のドローソフトを使って「WBS」を作成します。 階層は 3 階層以上にして下さい。アクティビティの数は 3 つ以上にして下さい。
作成した図を保存します。ファイル名は「WBS-1-4-1」とします。
次に任意のドローソフトを使って「アクティビティ間の依存関係図」を作成します。
作成した図を保存します。ファイル名は「WBS-1-4-2」とします。
次に任意の表計算ソフトを使って「WBS+ガントチャート」を作成します。
ガントチャートのクリティカルパスに★マークを記入します。
作成したシートを保存します。ファイル名は「WBS-1-4-3」とします。
チーム内で答え合わせをし、提出用の図を選びます。「WBS-1-4-3」は画像ファイルに変換して下さい。
指定場所に「WBS」「依存関係図」「WBS+ガントチャート」を貼り付けて下さい。


演習 1-5 (個人PC、30分): (案) 「二人で効率よく野菜炒めを作って食べる」ための WBS 、依存関係図、WBS+ガントチャートを作ってクリティカルパスを見つける
