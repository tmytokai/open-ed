4. ディープラーニングの実行


演習 2-2 (個人PC、20分): 画像に対してディープラーニングを実行してみましょう。

ニューラルネットワークの仕様は以下の通りです。

仕様:

(1) 入力層が N = 9 個、隠れ層が K = 5 個、出力層が M = 2 個のパーセプトロンで出来ている

(2) 教師信号は以下の 3x3 サイズ(つまり N = 9 )の 4 画像とする(つまり教師信号は L = 4 組)。 

    [
        # 画像1
        [
            1, 0, 1,
            1, 1, 1,
            1, 0, 1
        ],
        # 画像2
        [
            0, 1, 0,
            1, 0, 1,
            0, 1, 0
        ],
        # 画像3
        [
            0, 1, 0,
            1, 1, 1,
            0, 1, 0
        ],
        # 画像4
        [
            1, 0, 1,
            0, 1, 0,
            1, 0, 1
        ]
    ]

(3) 画像1と2はクラス1、画像3と4はクラス2に属するものとしてラベルを作成する(つまり M = 2 クラス)。

(4) 重みとバイアスは平均 0、標準偏差 0.1 の正規乱数で初期化する。

(4) 学習率は 0.01、学習回数は 20000 回とする。

テンプレートスクリプトファイル(template02-2.py)をダウンロードして下さい。
テキストと template02-1.py を参考にしながらスクリプトファイル内の ? の部分を上から穴埋めして行って下さい。
穴埋めが終わったら実行して下さい。
「スクリプトファイルの内容」と「表示結果」をドキュメントファイルに記述して保存してください。
