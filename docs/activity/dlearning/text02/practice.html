5. 未知入力データの分類


演習 3 (個人PC、30分): ディープラーニングで簡単な画像の分類をしてみましょう。

ニューラルネットワークの仕様は以下の通りです。

仕様:

(1) N = 9、K = 5, M = 4 とする。

(2) 学習データは以下の 3x3 サイズ(つまり K = 9 )の 4 画像とする。 

    [
        # 画像1
        [
            1, 0, 1,
            1, 1, 1,
            1, 0, 1
        ],
        # 画像2
        [
            0, 1, 0,
            1, 0, 1,
            0, 1, 0
        ],
        # 画像3
        [
            0, 1, 0,
            1, 1, 1,
            0, 1, 0
        ],
        # 画像4
        [
            1, 0, 1,
            0, 1, 0,
            1, 0, 1
        ]
    ]

(3) 各画像はそれぞれ別々のクラスに属するものとしてラベルを作成する(つまり M = 4 クラス)。

(4) 重みとバイアスは平均 0、標準偏差 0.1 の正規乱数で初期化する。

(4) 学習率は 0.01、学習回数は 20000 回とする。

(5) 未知入力データは以下とする。 

   [
       # 入力画像
       [
           1, 1, 0,
           1, 1, 1,
           1, 0, 1
       ]
    ]

テンプレートスクリプトファイル(template02-3.py)をダウンロードして下さい。
テキストと template02-2.py を参考にしながらスクリプトファイル内の ? の部分を上から穴埋めして行って下さい。
穴埋めが終わったら実行して下さい。
入力画像の値を変えると分類結果が変わる事を確認して下さい。
「スクリプトファイルの内容」と「表示結果」をドキュメントファイルに記述して保存してください。
