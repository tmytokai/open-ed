1. パーセプトロン

生物学で学んだように、動物の脳は多数(人間の大脳の場合140億個位)の「神経細胞」という細胞がシナプス結合されて出来ています。そこでまず研究者はこの「神経細胞」を数学的にモデル化した「パーセプトロン」を考案しました。

パーセプトロンは $n$ 入力 $1$ 出力の関数 $f()$ であり、以下の数式で表すことができます。

定義: パーセプトロンの式: \[ y = f \left ( \sum_{i=1}^{n} \{w_ix_i\} + b \right ) \]
$y$ : 出力
$w_i$ : $i$ 番目の入力に掛けられる重み(weight)
$x_i$ : $i$ 番目の入力
$b$ : バイアス

また関数 $f()$ の事を正式には「活性化関数」といいます。活性化関数の選び方には色々な流儀があるのですが、今回のアクティビティでは良く使われている「シグモイド関数」と「SoftMax」関数を用いたいと思います。

$w_i$ と $b$ は変数なので値は自由に決めても良いのですが、普通は教師データを使ってディープラーニングにより値を求めます。詳しくはディープラーニングのページで説明します。

2. 多層ニューラルネットワーク

次に研究者は「パーセプトロン」を多数結合して「多層ニューラルネットワーク」を考案しました。 「多層構造ニューラルネットワーク」とか「多層パーセプトロン」などとも呼ばれることもあります。 なお今回のアクティビティでは簡単のために、出力を入力にフィードバックしない「フィードフォワード・ニューラルネットワーク」だけを取り扱います。

多層ニューラルネットワークは「入力層」、「隠れ層(中間層とも言う)」、「出力層」ごとにレイヤー分けされた多層構造になっています(図2)。「入力層」、「出力層」は 1 層構造なのですが、「隠れ層」は 2 層以上の多層構造になる場合もあります。いずれにしろ、各層は多数のパーセプトロンが結合されて出来ています。なおパーセプトロンの事を「ユニット」と呼ぶこともあります。

図2. 多層ニューラルネットワークのグラフ:

多数のパーセプトロン(ユニット)が階層状に結合されて出来ている

ただしあまり隠れ層の層数を増やすとディープラーニングに必要な時間が膨大になりますので、今回のアクティビティでは中間層が 1 層だけで出来ている 3 層ニューラルネットワークを扱いたいと思います(図3)。

図3. 3層ニューラルネットワークのグラフ:

入力層が N 個、隠れ層(1層)が K 個、出力層が M 個のパーセプトロンで出来ているネットワーク

$w_{ij}$ ・・・ 前層のパーセプトロン No.$i$ の出力から、ある層のパーセプトロン No.$j$ への入力に掛けられる重み
$b_{j}$ ・・・ ある層のパーセプトロン No.$j$ のバイアス
