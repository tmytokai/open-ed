では早速 Keras を用いて MLP の構築と学習をしてみましょう。

まず今回は MLP として3層ニューラルネットワークを考えていました(図1)。

図1. 3層ニューラルネットワーク(再掲)

入力層が N 個、隠れ層が K 個、出力層が M 個のパーセプトロンで出来ている


この3層ニューラルネットワークを Keras を用いて構築する例が次のソース 1 です。
生の TensorFlow を使った時と異なり、とてもシンプルに 3 層ニューラルネットワークを構築出来ます。

ソース 1: Keras で3層ニューラルネットワークを構築する例 
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

# パーセプトロンの数
N = 3  # 入力層
K = 5  # 隠れ層
M = 2  # 出力層

# 入力信号
data = tf.constant( [[1, 2, 3]], dtype=tf.float32 )

# 3層ニューラルネットワークの構築
model = Sequential()
model.add( Dense(K, activation='tanh',input_dim=N) )
model.add( Dense(M, activation='softmax') )

# 出力
y = model.predict(data)

# モデルの概要表示
model.summary()

print('入力')
print( data.numpy() )
print('\n出力')
print( y )

では各行の意味について順を追って説明していきたいと思います。

まず最初の 

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

でこれから使う Sequential クラスと Dense クラスの import を行っています。

パーセプトロンの数や入力信号の定義部分は飛ばし、次の 

model = Sequential()

の行では Sequential クラスのインスタンスである model を作成しています。
この様にニューラルネットワークの構造(モデル)として Sequential モデルを選ぶと add メソッドを使って各層を順々に積み重ねることが出来るようになります。
例えば次の 

model.add( Dense(K, activation='tanh',input_dim=N) )

の行では add メソッドを使って model に隠れ層のインスタンスを追加しています。
ここで Dense は「全結合層」を意味するクラスであり、引数の K はパーセプトロンの数、activation='tanh' は活性化関数として双曲線正接(tanh)を使う事、input_dim=N は入力信号の次元として N を指定する事を意味します。

さらに続けて、次の 

model.add( Dense(M, activation='softmax') )

の行でも add メソッドを使って model に出力層のインスタンスを追加しています。
ただし出力層のパーセプトロン数として M、活性化関数として softmax 関数を指定しています。
なお最初に add する層以降では input_dim を指定する必要はありません(自動計算してくれるので)。

これで3層ニューラルネットワークが出来たので、次の行の 

y = model.predict(data)

の様に predict メソッドを使って入力信号 data に対する出力 y が求められます。

また、次の 

model.summary()

の行ではモデルの概要を表示しています(図2)。 

図2. リスト1で構築したニューラルネットワークの概要

Model: "sequential_10"
_________________________________________________________________
Layer (type)                 Output Shape              Param #   
=================================================================
dense_20 (Dense)             (None, 5)                 20        
_________________________________________________________________
dense_21 (Dense)             (None, 2)                 12        
=================================================================
Total params: 32
Trainable params: 32
Non-trainable params: 0

この概要を見ると、確かに K=5 の隠れ層と M=2 の出力層からなる3層ニューラルネットワークになっている事が分かります(上から下に向かって信号が流れます)。

なお Param の行にある数字は各層のパラメータの個数です。
図1より、隠れ層の重みの数は N*K 個でバイアスの数は K 個なので、隠れ層のパラメータ数は N*K+K = 3*5+5 = 20 個となります。
同様に、出力層の重みの数は K*M 個でバイアスの数は M 個なので、出力層のパラメータ数は K*M+M = 5*2+2 = 12 個となります。
