Python の関数オブジェクト(def)を利用してオペレーション(以下 OP)を自作することができます。

以下は (x+y)*z の演算を行う hoge という名の OP を作成する例です。
内部で足し算 OP と掛け算 OP を使用しています。

OP の自作例: 
#coding: UTF-8
import tensorflow as tf
sess = tf.Session()

#自作 OP   (x+y)*z
def hoge(x, y, z):
  op_add = tf.add( x, y )
  op_mul = tf.multiply( op_add, z )
  return op_mul

op_hoge = hoge( 1, 2, 3 )
result = sess.run( op_hoge )
print( result )

# 表示結果
