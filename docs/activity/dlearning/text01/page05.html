Python の関数オブジェクト定義を利用してオペレーションを自作することができます。

自作オペレーションの作り方:

書式: 

@tf.function
def オペレーション名(引数)
  (演算)
  return 戻り値

例えば、以下の例では (x+y)*z の演算を行う myOP(x,y,z) というオペレーションを作成しています。

オペレーションの自作例: 
import tensorflow as tf

#自作オペレーション : myOP(x,y,z) = (x+y)*z
@tf.function
def myOP(x, y, z):
  tmp1 = tf.math.add( x, y )
  tmp2 = tf.math.multiply( tmp1, z )
  return tmp2

# (注)
# @tf.function
# def myOP(x, y, z):
#   return tf.math.multiply( tf.math.add( x, y ), z )
# 
# と書いても良いが、分かりにくいので tmp1、tmp2 という一時的なテンソルを使っている

A = tf.constant( [1,2], dtype=tf.float32 )
B = tf.constant( [3,4], dtype=tf.float32 )
C = tf.constant( [5,6], dtype=tf.float32 )
D = myOP(A,B,C)
print( D.numpy() )

# 表示結果
# [20. 36.]
