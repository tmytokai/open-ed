TensorFlow(テンソルフロー)は Google が開発してオープンソースとして公開しているディープラーニング(機械学習)用ライブラリです。
検索での音声認識、Googleフォトでの画像認識、Google翻訳などの自社サービスで実際に使われています。

TensorFlow は具体的には高速で数値演算や行列演算を行うための演算ライブラリであり Python や C++ から利用することが出来ます。
その高速演算能力をディープラーニングに生かして画像・音声の認識や検索、翻訳などで利用されています。

TensorFlow に関して詳しい事は自分で調べてもらうことにして、早速 TensorFlow でハローワールドを動かしてみましょう。
なお今回のアクティビティではプログラミング言語として python を使います。

TensorFlow のハローワールド:

python を起動して以下のコードを実行すると 123 と表示される。 

#coding: UTF-8
import tensorflow as tf
sess = tf.Session()

op_const = tf.constant( 123 )

result = sess.run( op_const )
print( result )

# 表示結果
# 123

各行の意味についてはこれから順を追って説明していきたいと思います。
