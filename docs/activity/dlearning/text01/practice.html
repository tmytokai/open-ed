5. OP ノードの自作


演習 8 (個人PC、10分): 自作 OP ノードを作ってみましょう。

仕様は以下の通りです。

仕様:

(1) 名前は mynode とする

(2) 引数は x、y、z の 3 つ(いずれも行列とする)

(3) ( x + y ) と z の行列積を出力する

ホワイトボードに mynode OP ノードのデータフローグラフを描いて下さい。
データフローグラフを描いたらファシリテータの確認を受けて下さい。
データフローグラフの写真を撮ってドキュメントファイルに貼りつけて下さい。
スクリプトを作るためテキストエディタを開き、先頭に「#coding: UTF-8」を追加して下さい。
データフローグラフを見ながら mynode OP ノードを作成して下さい。
定数 OP ノードで 3 x 2 行列を 2 つ、 2 x 4 行列を作り(中身は適当)、それらを順に x, y, z として mynode に渡して実行した結果を表示するスクリプトを書いて下さい。
「1-8.py」で保存し実行して下さい。
「データフローグラフの写真」が貼ってあることを確認し、「スクリプトファイルの内容」と「表示結果」をドキュメントファイルに記述して保存してください。
