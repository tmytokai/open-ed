5. オペレーションの自作


演習 1-8 (個人PC、10分): 自分の OP を作ってみましょう。

作成する OP の仕様は以下の通りです。

仕様:

(1) 名前は mynode とする

(2) 引数は x、y、z の 3 つ(いずれも行列)

(3) ( x + y ) と z の行列積を出力する

スクリプトを作るためテキストエディタを開いて下さい。
mynode を作成して下さい。
3 x 2 行列「op_const_x」、3 x 2 行列「op_const_y」、2 x 4 行列「op_const_z」を定数 OP で作って下さい。中の値は適当で結構です。
op_const_x〜z を順に mynode に渡し、結果を result に代入して表示して下さい。
「DL-1-8.py」の名前で保存し実行して下さい。
