目的

あなたが TensorFlow / Keras を使ってディープラーニングが出来る技術者になること

必要性

AI の中核技術であるディープラーニングは IoT において必須となる技術です。
センサーによって集められる大量のデータを解析するために、ディープラーニングについて技術者は学ぶ必要があります。

進め方

上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
このアクティビティの具体的な進め方についてはこちらを参照して下さい。
