乱数は Math クラスを使って取得できます。

0から1までの実数の乱数を取得したい場合は次のように Math.random() を使います。

ソース1: 0 から 1 までの実数の乱数 
public class Main{
    public static void main(String[] args) {
        double a = Math.random();
        System.out.println( a );
    }
}

従って A 以上 B 以下の整数の乱数を取得したい場合は次のように書きます。

ソース2: A 以上 B 以下の整数の乱数 
public class Main{
    public static void main(String[] args) {
        int A = 5;
        int B = 8;
        int c = A + (int)(Math.random()*((B-A)+1));
        System.out.println( c );
    }
}
