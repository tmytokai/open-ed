Java も、C言語などと同じ様にまずソースコードをコンパイルしてバイナリファイル(正確には Java バイトコードと呼びます)を作る必要があります。
なお一般に Java のソースコードの拡張子は「.java」で、バイナリファイルの拡張子は「.class」となります。

さて Java のコンパイラには色々な種類がありますが、今回のアクティビティでは「javac」を使います。
なお Java のプログラムは「 Java 仮想マシン」と呼ばれる仮想のコンピュータ上で動かすという仕組みになっているので、javac でコンパイルして作ったバイナリファイルは「そのままでは実行できません」。
このJava仮想マシンにも色々な種類がありますが、今回のアクティビティでは「java」を使います。

では早速ソース 1 に示した Hello World のソースコードを使って、具体的にコンパイルと実行をしてみましょう。

ソース1: Hello World 
public class Main{
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

ここで 2 行目の「public static void main(String[] args)」が Java 言語のメイン関数になります。

※ 1 行目の「public class Main」の説明はとりあえず一旦置いておきます

そしてメイン関数の中で文字列 "Hello World" を画面に表示しています。
なお Java では関数のことを「メソッド」と呼びますので、以降はメイン関数を「メインメソッド」と呼ぶことにします。

ではこのソース1を以下の手順でコンパイル＆実行して下さい。

「Main.java」という空のテキストファイルを新規作成する
そのファイルにソース1の内容をコピーして上書き保存する
「javac Main.java」 でコンパイルしてバイナリファイル「Main.class」を作る
「java Main」 で実行する ※ 拡張子 .class は必要ありません
