Java も、C言語などと同じ様にまずソースコードをコンパイルしてバイナリファイル(正確には Java バイトコードと呼びます)を作る必要があります。
一般に Java のソースコードの拡張子は「.java」で、バイナリファイルの拡張子は「.class」です。

さて Java のコンパイラには色々な種類がありますが、今回のアクティビティでは「javac」を使います。

コンパイルが終わったら次はプログラムを実行しますが、javac でコンパイルして作ったバイナリファイルは「そのままでは実行できません」。
Java のプログラムは「Java 仮想マシン」と呼ばれる仮想のコンピュータ上で動かすという仕組みになっています。
この Java 仮想マシンにも色々な種類がありますが、今回のアクティビティでは「java」を使います。

では早速ソース 1 に示した Hello World のソースコードを使って、具体的にコンパイルと実行をしてみましょう。

ソース1: Hello World 
public class Main{
    public static void main(String[] args) {  // メインメソッド
        System.out.println("Hello World");
    }
}

1 行目の「public class Main」についてはこちらで説明しますが、ここでは一旦置いておきます

2 行目の「public static void main(String[] args)」が Java 言語のメイン関数になります。
このメイン関数の中で文字列 "Hello World" を画面に表示しています。
なお Java では関数のことを「メソッド」と呼びますので、以降はメイン関数を「メインメソッド」と呼ぶことにします。

ではこのソース1を以下の手順でコンパイル＆実行して下さい。

ソース 1 のコンパイル＆実行手順

「Main.java」という空のテキストファイルを新規作成する
そのファイルにソース1の内容をコピーして上書き保存する
「javac Main.java」 でコンパイルしてバイナリファイル「Main.class」を作る
「java Main」 で実行する ※ 拡張子 .class は必要ありません
