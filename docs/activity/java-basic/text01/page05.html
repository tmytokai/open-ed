Java の演算処理はほぼ C 言語の演算処理とほぼ同じです。
例えば四則演算は +-*/ ですし、%は剰余演算、++と--はインクリメントとデクリメントになります。

また += や *= などの代入演算子や、== などの比較演算子や、>> などのビット演算子も C 言語と同じ様に使えます。

ただし String 型の変数同士の足し算はこちらで説明したように「文字列の連結」になりますのでご注意下さい。
またString 型の比較は == ではなくて「変数名.equals(文字列)」の方を使って下さい。
