Java でファイル入出力を行う方法は種類が多くとても複雑です。
今回はとりあえず使うことを最優先にし、細かい内容は省いて説明していきたいと思います。
もし詳しく知りたい人は自分で調べてみて下さい。

まずファイル出力はソース 1 の様におこないます。

ソース1: ファイル出力の例 
import java.io.PrintWriter;
import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class Main{
    public static void main(String[] args) {
       try {
            PrintWriter writer = new PrintWriter( new BufferedWriter( new FileWriter( "hoge.txt" ) ) );
            int a = 1;
            int b = 2;
            writer.println( "足し算" );
            writer.println( a + "足す" + b + "は" + (a+b) );
            writer.close();
        }
        catch ( IOException e ) {
            System.out.println( e );
        }
    }
}

上の例ではtryの次のやたら長い行でファイルを書き込みモードでオープンしています。
今回は hoge.txt をオープンして writer というファイル書き込み処理を行うインスタンスを作成しています(あくまで例なのでインスタンスの名前は writer でなくても良いです)。

なおファイルをオープンする時は try のカッコ中に入れないとエラーがおきてコンパイルできません。
また実行時にファイルがオープン出来ない時は例外が生じ、 catch の中にジャンプしてエラー内容を表示します。

一度オープンすれば、あとは画面表示の時と同様に writer.println() でファイルに書き込みができます。

最後に writer.close() でファイルをクローズするのを忘れないで下さい。

それでソース 1 をコンパイルして実行すると以下の内容の hoge.txt が出来上がります。

hoge.txt の内容 
足し算
1足す2は3
