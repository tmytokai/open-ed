1. ファイル出力


演習 3-1 (個人): ソース1を修正して、次の仕様を満たすプログラムを作成しましょう。 

(1) キーボードから整数を入力して int a に代入する

(2) キーボードから整数を入力して int b に代入する

(3) 以下の文字列を hoge.txt に出力する

aは(aの数値)
bは(bの数値)
足すと(a+bの値)です

結果:

実行してキーボードから 12 と 23 を入力すると hoge.txt の内容は次のようになる

aは12
bは23
足すと35です

フォルダ「JAVABSC_3_1」を新規作成し、その中に移動します

ソース1をコピーし、解答します。
コンパイルして実行します。

2. ファイル入力

演習 3-2 (個人): ソース1を修正して、次の仕様を満たすプログラムを作成しましょう。 

(1) hoge.txt を作成し、行を分けて整数を3つ保存する

(2) hoge.txt を読み込んで、3つの数字の合計値を画面に出力する

※ 文字列を int 型に変換するためには Integer.parseInt( 文字列 ) を使う

結果:

hoge.txt の中身が

12
34
56

の時、実行すると画面には

102

と表示される

ソース1をコピーし、解答します。
コンパイルして実行します。
