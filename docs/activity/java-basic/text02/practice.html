演習 2-3 (個人、30分): 算術例外を試してみましょう。 

結果:

0を入力すると
算術例外発生: java.lang.ArithmeticException: / by zero

1を入力すると
入出力例外発生: java.io.FileNotFoundException: hoge.txt (No such file or directory)

演習 2-2 で作った「JAVABSC_2_2」をフォルダごとコピーし、フォルダ名を「JAVABSC_2_3」とします。
「JAVABSC_2_3」フォルダ内の hoge.txt を消します( fuga.txt は残したままにします )。
try 文が書いてある行と hoge.txt をオープンしている行の間に以下の処理を追加します。

  • int 型変数 a にキーボードから整数を代入する
  • 1/a の計算結果を画面に表示する

IOException の catch 処理の下に ArithmeticException の catch 処理を追加します。なお println の中身は「"算術例外発生: "+e」として下さい。
コンパイルと実行をし、0 を入力したら算術例外、0 以外を入力したら入出力例外が起きることを確認して下さい。
