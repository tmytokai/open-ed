前のページのソース 1 に出てきた IOException クラスは Java に初めから備わっている例外クラスでしたが、これ以外にも Java には例外の種類に合わせて様々な例外クラスが用意されています。
Java に初めから備わっている例外クラス、あるいは例外処理そのものの事を「標準例外」と言います。

なお「標準例外」は別に Java だけの機能ではなくて、オプジェクト指向プログラミング言語には大抵備わっています。
標準例外の種類は言語によって変わりますが、以下の標準例外は大抵の言語に備わっています(もちろん他にも沢山あります)。

  • 入出力例外: ファイル等の入出力に失敗した
  • 算術例外: ゼロで数字を割ろうとした等、演算に失敗した
  • NULL ポインター例外: NULL ポインターにアクセスしようとした

Java では上の 3 つの標準例外は次の例外クラスに割当たっています。

  • 入出力例外 → IOException
  • 算術例外 → ArithmeticException
  • NULLポインター例外 → NullPointerException
