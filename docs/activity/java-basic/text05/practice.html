3. forEach 文

演習 5-4 (個人、? 分): 以下のソースコードで使われている通常の for 文を forEach 文を使うように書き換えて下さい。 

import java.util.ArrayList;

public class Main{

    // (注意) Javaの仕様上、ラムダ式の内部ではローカル変数の値を書き換えられないので static int にしています
    static int goukei = 0;

    public static void main(String[] args) {

    	ArrayList<Integer> list = new ArrayList<Integer>();

    	list.add( 1 );
    	list.add( 2 );
    	list.add( 3 );

        // ここから演習
        // forEach 文を使って合計を求める
        for( int i = 0; i < list.size(); ++i ){
            goukei += list.get(i);
        }

        System.out.println( goukei ); 
    }
}

結果:

6
