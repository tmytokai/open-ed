3. forEach 文


演習 5-3 (個人): 以下のソースコードで使われている拡張 for 文を forEach 文を使うように書き換えて下さい。 

import java.util.ArrayList;

public class Main{

    // Javaのラムダ式内ではローカル変数の値を書き換えられないので static int にしている
    static int goukei = 0;

    public static void main(String[] args) {

    	ArrayList<Integer> list = new ArrayList<Integer>();

    	list.add( 4 );
    	list.add( -3 );
    	list.add( 5 );

        // ここから演習
        // forEach 文を使って合計を求める
        for( int a : list ){ goukei += a; }

        System.out.println( goukei ); 
    }
}

結果:

6
