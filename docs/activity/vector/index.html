目的

あなたがベクトルと多様体を理解出来る技術者になること

必要性

電気電子機器を設計するための要の技術である電磁気学ではベクトルや多様体の知識が必要とされます。
電磁気学をきちんと理解して製品開発に活かすために、ベクトルと多様体について技術者は学ぶ必要があります。

進め方

もし上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
またこのアクティビティの具体的な進め方についてはこちらを参照して下さい。
