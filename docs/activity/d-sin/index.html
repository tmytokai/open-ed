目的

あなたが時間領域ディジタルサイン波の性質を理解し、グラフを描ける技術者になること

必要性

時間領域ディジタルサイン波は時間領域ディジタル信号に対するスペクトル解析のベースとなる信号です。
従ってスペクトル解析を学ぶために、ディジタルサイン波の性質とグラフの描き方を技術者は学ぶ必要があります。

進め方

もし上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
またこのアクティビティの具体的な進め方についてはこちらを参照して下さい。
