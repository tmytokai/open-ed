新たに「Hanako」シーンを作成する

メニューから「シーン → 新しい継承シーン」を選択し、「actor.tscn」を選ぶ

シーンドックの「Actor」を「Hanako」に変更する

シーンドックの「Hanako」を右クリックして「スクリプトを拡張」を選び、Hanakoのスクリプトファイル(hanako.gd)を作成する。

Sprite2Dノードをクリックし、ファイルシステムドッグ内の「img/HANAKO.jpg」を「Texure」の「<空>」にドッグする

2D画面を表示し、画像(50x57)が表示されていることを確認する

CollisionShape2Dノードをクリックし、「Shape」を「CapsuleShape2D」にし、当たり判定の領域を調整する

Ctrl+S を押してHanakoシーン(hanako.tscn)を保存する

タブでMyWorldシーンに切り替える

スクリプト画面に切り替えて my_world.gd を開く

以下のソースコードを追加する

ソース 1: 開始時にHanakoを(200,200)の位置に表示 
# インスタンス化するシーンを選択
var hanako_scene = preload("res://hanako.tscn")

# 開始時に1度だけ実行される関数
func _ready():
	# Hanako シーンをインスタンス化
	var hanako = hanako_scene.instantiate()
	# インスタンス化したHanakoの座標をセット
	hanako.position = Vector2(200,200)
	# MyWorldシーンにHanakoを追加
	add_child(hanako)


MyWorldシーンを開いているのを確認してから「現在のシーンを実行」ボタンを押すとHanakoが表示されるのを確認する
