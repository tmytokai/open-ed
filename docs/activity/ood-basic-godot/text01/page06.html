ソース 1: "a"を押したら Label にHelloと表示、"s"を押したら消す 
	# Input.is_action_just_pressed 関数でキーを押しているか「1度だけ」チェック出来る
	if Input.is_action_just_pressed("a"):
		# $"../Label" でLabelノードを参照できる
		$"../Label".text = "Hello"
		$"../Label".show()
	if Input.is_action_just_pressed("s"):
		$"../Label".hide()
