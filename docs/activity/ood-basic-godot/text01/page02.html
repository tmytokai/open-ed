テンプレートをインポートする

メニューから「シーン → 新しい継承シーン」を選択し、「world.tscn」を選ぶ

シーンドックの「World」を「MyWorld」に変更する

シーンドックの「MyWorld」を右クリックして「スクリプトを拡張」を選び、MyWorldのスクリプトファイル(my_world.gd)を作成する。

Sprite2Dノードをクリックし、ファイルシステムパネル内の「img/MyWorld.jpg」を「Texure」の「<空>」にドロップする

2D画面を表示し、背景画像(800x400)が表示されていることを確認する

メニューから「プロジェクト → ブロジェクト設定 → 一般 → 表示 → ウィンドウ」を選択し、「ビューポートの幅」を800、「ビューポートの高さ」を400にすると表示領域のサイズが背景画像と同じになる

Ctrl+s を押してMyWorldシーン(my_world.tscn)を保存する
