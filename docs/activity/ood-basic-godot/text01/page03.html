メニューから「シーン → 新しい継承シーン」を選択し、「actor.tscn」を選ぶ

シーンドックの「Actor」を「TARO」に変更する

シーンドックの「TARO」を右クリックして「スクリプトを拡張」を選び、TAROのスクリプトファイル(taro.gd)を作成する。

Sprite2Dノードをクリックし、ファイルシステムパネル内の「img/TARO.jpg」を「Texure」の「<空>」にドロップする

2D画面を表示し、画像(50x57)が表示されていることを確認する

CollisionShape2Dノードをクリックし、「Shape」を「CapsuleShape2D」にし、当たり判定の領域を調整する

Ctrl+S を押してTAROシーン(taro.tscn)を保存する

タブでMyWorldシーンに切り替える

シーンパネルの「MyWorld」を右クリックして「子シーンをインスタンス化 → taro.tscn」を選ぶとTAROが表示される

TAROをドラッグして適当な位置に移動する

同様にしてTAROを3人表示する

「TARO3」を右クリックして「削除」を選ぶとTARO3が消える
