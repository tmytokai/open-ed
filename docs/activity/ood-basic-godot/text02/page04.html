コーディング

スプリントバックログの内容に沿って、以下の様にコーディングします。

新規プロジェクトを作成する

テンプレートのフォルダ名を「GF-2」に変更します。
「今すぐ編集を始める」のチェックを外して「GF-2」をインポートします。
「名前の変更」を押して名前を「GF-2」に変更します。
プロジェクトを開きます。


MyWorld を表示する

「World」シーンを継承して「MyWorld」シーンを作成します。画像(MyWorld.jpg)もセットして下さい。

メニューから「プロジェクト → ブロジェクト設定 → 一般 → 表示 → ウィンドウ」を選択し、「ビューポートの幅」を800、「ビューポートの高さ」を450にセットしてください。


TARO と HANAKO を初期配置して表示する

「Actor」シーンを継承して「TARO」シーンを作成します。画像(TARO.png)もセットして下さい。
「Actor」シーンを継承して「HANAKO」シーンを作成します。画像(HANAKO.png)もセットして下さい。
「MyWorld」シーンのスクリプト(my_world.gd)を開き、_ready()関数で「TARO」を初期配置して下さい。座標は(0,0)です。
同様に「HANAKO」を初期配置して下さい。座標は(800,450)です。


TARO をカーソルキーで上下左右に動かす

「TARO」シーンのスクリプト(taro.gd)を開き、_process関数内に矢印キーで上下左右に動かすプログラムを記述して下さい。 スピードは100です。


HANAKO をカーソルキーで上下左右に動かす

「HANAKO」シーンのスクリプト(hanako.gd)を開き、_process メソッド内に矢印キーで上下左右に動かす(ただし上下左右は反転)プログラムを記述して下さい。 スピードは100です。


テスト

コーディングが終わったらテストします。
MyWorldシーンを開いているのを確認してから「現在のシーンを実行」ボタンを押して動作確認して下さい。
