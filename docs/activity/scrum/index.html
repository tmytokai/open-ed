目的

あなたがスクラム開発を使ってソフトウェアを迅速に開発できる技術者になること

必要性

スクラム開発はアジャイルなソフトウェア開発手法のひとつです。
チームによる開発を円滑かつ高速に進めるため、スクラム開発について技術者は学ぶ必要があります。
