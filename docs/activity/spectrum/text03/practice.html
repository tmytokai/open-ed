演習3-1 (個人、45分): 自分の声をスペクトル解析してみましょう。

マイクを PC の端子に繋ぎます。
マイクの設定を開いて音量調整をします。
レコーダーを開きます。
「2 秒以上」録音し、ダウンロードフォルダに音声ファイル( out.wav ) をダウンロードします。
out.wav を試しに再生し、もし音が小さい場合は録音し直します。
out.wav を演習フォルダに移動して「 SPEC-3-1.wav 」に名前を変更します。
テンプレートソース(SPEC-3-1.cpp)をダウンロードします。
テンプレートファイルの記述に従ってプログラムを完成させて下さい。
コンパイルして実行すると「 SPEC-3-1.csv 」という CSV ファイルと、1秒地点から0.5秒分の音声を切り出した「 SPEC-3-1-out.wav 」が出来ます。
SPEC-3-1.csv を表計算ソフトで開き、一行目に空行を挿入し、最初の列の先頭に「Hz」、次の列の先頭に「振幅スペクトル」、その次の列の先頭に「位相スペクトル」と入力します。
振幅スペクトルのグラフを挿入します。グラフの種類は散布図、点「無し」、線あり、平滑化「無し」とします。

※1 今回は 8000 Hz までのスペクトルを出力しています。
※2 行数が多いので範囲選択すると表計算ソフトが固まる時があります。Ctrlキーを押しながら「Hz」列と「振幅スペクトル」列をクリックして列の選択をして下さい。

同様に位相スペクトルのグラフを挿入します。グラフの種類は散布図、点「無し」、線あり、平滑化「無し」とします。
シートを保存します。ファイル名は「SPEC-3-1」として下さい(拡張子は表計算ソフトによる)。
振幅スペクトルを解析し、自分の声の周波数的な特徴を見つけて下さい。
チーム内で答え合わせをして下さい。
各自が指定場所に「振幅スペクトル」と「自分の声の周波数的特徴」を記入して下さい。
