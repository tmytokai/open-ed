演習 1-1 (チーム、10分): 電磁波はエネルギーでもあり粒(光子)でもある特殊なサイン波で、通常は周波数 [Hz] でなくて波長 [m] を使って電磁波の種類が分類されています。

その変換式は「波長 = 光速度 / 周波数」ですので、波長と周波数は反比例関係にあります。

代表的な電磁波である

  • 「赤外線」
  • 「紫外線」
  • 「ガンマ線」
  • 「可視光線の赤色」
  • 「可視光線の青色」
  • 「X線」
  • 「電波」

の波長と周波数帯をチームみんなで手分けして Web 上等で調べ、指定場所に波長の短い電磁波から昇順に(つまり周波数が高い電磁波から降順に)並べて書いてください。


演習 1-2 (チーム、20分): スペクトル解析の実際例を調べてみましょう。

  • 「人間や動物の声をスペクトル解析すると何が分かるのか」
  • 「物質から出てくる光や吸収される光をスペクトル解析すると何が分かるのか」
  • 「自分が好きな星(太陽や惑星や恒星)の光のスペクトル図、およびその図から何が分かるのか」
  • 「スマホから出てくるブルーライトのスペクトル図、および何故ブルーライトが目に悪いのか」
をチームみんなで手分けして Web 上等で調べ、指定場所に調べた内容を書いて下さい。
