演習 1-1 (個人): 太陽や蛍光灯などの光をスペクトル解析すると、光には様々な電磁波が含まれていることが分かります。
例えば蛍光灯のカタログなどをみるとスペクトル分布によって蛍光灯の光色に違いがあることが分かります。

この電磁波はエネルギーでもあり粒(光子)でもある特殊なサイン波で、通常は周波数 [Hz] でなくて波長 [m] を使って電磁波の種類が分類されています。
その変換式は「波長 = 光速度 / 周波数」ですので、「波長と周波数は反比例関係」にあります。

そこで代表的な電磁波である

  • 「赤外線」
  • 「紫外線」
  • 「ガンマ線」
  • 「可視光線の赤色」
  • 「可視光線の青色」
  • 「X線」
  • 「電波」

の波長を調べ、指定した場所に「各電磁波の波長」と「波長の短い電磁波から昇順に(つまり周波数が高い電磁波から降順に)並べた結果」を記入して下さい。
