演習 1-1 (チーム、10分): 電磁波はエネルギーでもあり粒(光子)でもある特殊なサイン波で、通常は周波数 [Hz] でなくて波長 [m] を使って電磁波の種類が分類されています。

その変換式は「波長 = 光速度 / 周波数」ですので、波長と周波数は反比例関係にあります。

代表的な電磁波である

  • 「赤外線」
  • 「紫外線」
  • 「ガンマ線」
  • 「可視光線の赤色」
  • 「可視光線の青色」
  • 「X線」
  • 「電波」

の波長と周波数帯をチームみんなで手分けして Web 上等で調べ、指定場所に波長の短い電磁波から昇順に(つまり周波数が高い電磁波から降順に)並べて書いてください。


演習 1-2 (チーム、20分): スペクトル解析の実際例を調べてみましょう。以下の項目についてチームみんなで手分けして調べてください(1人1件)。

  • 「人間や動物の声のスペクトル図を検索して貼り付ける。また、その声のスペクトル解析から何が分かるのか調べて書く」
  • 「物質から出てくる光や吸収される光のスペクトル図を検索して貼り付ける。また、その光のスペクトル解析から何が分かるのか調べて書く」
  • 「星(太陽や惑星や恒星)の光のスペクトル図を検索して貼り付ける。また、その光のスペクトル解析から何が分かるのかを調べて書く」
  • 「スマホから出てくるブルーライトのスペクトル図を検索して貼り付ける。また、その光のスペクトル解析から何故ブルーライトが目に悪いのかを調べて書く」
