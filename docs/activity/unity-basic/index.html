目的

Unity を使ってゲーム開発が出来る技術者になること

必要性

Unity は PC (Windows、Mac、Linux)に加えスマートデバイスや Web ブラウザ上で動作するゲームを作成できるクロスプラットホームなゲーム開発エンジンです。
Unity は世間一般で広く利用されているので、Unity について技術者は学ぶ必要があります。
