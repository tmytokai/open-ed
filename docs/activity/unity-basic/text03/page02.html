1. 詳細設計

この工程では、背景となる「MyWorld」オブジェクトを作ってカメラ調整をします。

詳細設計:

  • 背景画像は任意とします。
  • 画面比率は 5:4 とします。

2. 演習内容

準備:

Unity エディタ を起動します。
新規プロジェクトを作成します。ブロジェクト名は「UNITY-BASIC3」として下さい。


コーディング 1: 「MyWorld」オブジェクトの作成

背景画像を用意し「MyWorld」に名前変更して下さい(※ 拡張子は変更しないよう注意)。
画像を「MyWorld」オブジェクト化して下さい。


コーディング 2: カメラの調整

カメラを調整して下さい。画面比率は 5:4 とします。


テストとデバッグ:

実行ボタンを押して動作確認して下さい。
上手く動作しなかった場合は実行ボタンを押して動作を止めてデバッグして下さい。
