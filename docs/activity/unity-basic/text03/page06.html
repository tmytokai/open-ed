ToDo
HANAKO を表示する

TARO と HANAKO がすれ違えるようにする

コーディング

スプリントバックログの内容に沿って、以下の様にコーディングします。

HANAKO を表示する

画像を用意し「HANAKO」に名前変更して下さい(※ 拡張子は変更しないよう注意)。
画像をアセットに登録し、更にゲームオブジェクト化して下さい。
Inspector ウィンドウの Transform の Position の x と y で HANAKO の初期位置をセットして下さい。座標は(3.7,-1.9)です。


HANAKO を上下左右に動かす

「HANAKO」を選択し、Inspector で Rigidbody 2D をアタッチして設定します。重力(Gravity Scale)を 0 にし、他のオブジェクトとぶつかっても回転しない様に Constraints → Freeze Rotation → Z にチェックを入れて下さい。
「HANAKO」スクリプトを新規作成し、「HANAKO」オブジェクトにアタッチして下さい。
「HANAKO」スクリプトをエディタで開き、 Update メソッドの下に void FixedUpdate() メソッドを追加して下さい。
FixedUpdate メソッド内に「上下左右に動かす」プログラムを記述して下さい。ただし上下左右を反転します(上を押したら下、下を押したら上、左を押したら右、右を押したら左に移動)。移動速度は x 方向、y 方向ともに 2f です。


HANAKO が画面外に出ないようにする

「HANAKO」を選択し、Inspector で Box Collider 2D をアタッチして下さい。衝突の範囲はデフォルトのままで結構です。


TARO と HANAKO がすれ違えるようにする

「TARO」を選択し、Inspector でレイヤーを「Player」にしてください。
「HANAKO」を選択し、Inspector でレイヤーを「Player」にしてください。
Layer Collision Matrix の設定を開き、「Player」-「Player」 間のチェックを外して下さい。
