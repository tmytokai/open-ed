1. スプリント プランニング

詳細設計:

このスプリントでは、タイムアップ処理を行います。

  • タイムアップ処理は MyWorld スクリプトで行います。
  • ゲーム開始からの経過秒数を表示します。
  • 経過秒数表示用の Text オブジェクトの名前は「TimeText」とします
  • 制限時間は任意とします。
  • タイムアップしたら MessageTextに メッセージを表示してゲームオーバーにします。
  • メッセージの内容は任意とします。

スプリント バックログ:

  1. 経過秒数表示用の「TimeText」の作成
  2. 秒数表示とタイムアップ処理

2. 開発作業

準備:

プロジェクト「UNITY-2」に対して作業を続けます。


スプリントバックログ 1: メッセージ表示用の「経過秒数表示用の「TimeText」の作成」の作成

「MyCanvas」の下に「TimeText」という Text 型オブジェクトを新規作成します。
「TimeText」の位置やフォントサイズ等を適宜調整します。
調整が終わったら文字列(New Text)は消しておきます。


スプリントバックログ 2 : 秒数表示とタイムアップ処理

「MyWorld」スクリプトをエディタで開き、 using 部に「using UnityEngine.UI;」を追加します。
Update メソッド内に、「経過秒数を TimeText に表示し、秒数が制限時間を超えたら MessageText にメッセージを表示してプログラムを停止する」プログラムを記述して下さい。 制限時間は任意です。
なおゲーム開始からの経過秒数は、数値(float)なら Time.time 、文字列(String)なら Time.time.ToString() で取得できます。


テストとデバッグ:

実行ボタンを押して動作確認して下さい。
上手く動作しなかった場合は実行ボタンを押して動作を止めてデバッグして下さい。


3. リリース

チーム内で答え合わせをし、提出用のスクリプト(MyWorld.cs)を選んで下さい。
指定場所に「ソースコード」を貼り付けて下さい。
