この学習項目では 2D のジャンプアクションゲームを作成してみましょう。
なお画像として学習項目[2] に引き続き いらすとや様の作品を利用させて頂いてます。

まず要求定義を行って要求仕様書を作成します。
今回は以下の様な要求仕様書を作成しました。

要求仕様書

タイトルTARO と OBAKE
要求 ・暇つぶし用ミニゲーム
・ライト層向け
・2 次元ジャンプアクションゲーム
・PC 用
・画面上に TARO というプレイヤーキャラと OBAKE という複数の敵キャラと障害物を表示する
・ユーザーは TARO を動かせる
・TARO はボールを投げて OBAKE を退治できる
・OBAKE はランダムに動きまわる
・全ての OBAKE を倒したらゲームクリア
・TARO が OBAKE に触れたらゲームオーバー
