演習 5-1 (個人): 自動車シミュレータを改造して何かゲームを作ってみましょう

ソースコードをダウンロードして展開します。

※ タイヤのモデルとしてこちらのアセットを使用しています

Scenes → SampleScene を開きます。
実行ボタンを押して動作確認します。

※操作方法
ハンドル操作 → 左右ボタン (左右カーソルキー)
前進 → Jump ボタン (スペースキー)
ブレーキ / 後退 → Fire2 ボタン (左Alt / Option)

後は好きに改造して下さい。
指定場所に「改造後のソースコード」を提出して下さい。
