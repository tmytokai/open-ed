Unity エディタの再生をスクリプトから停止したい場合は次のように書きます。

Unity エディタの再生を停止 
using の所に

#if UNITY_EDITOR 
using UnityEditor; 
#endif

を追加してから

#if UNITY_EDITOR 
EditorApplication.isPlaying = false;
#endif

またビルドして作成したアプリケーションをスクリプトから停止したい場合は次のように書きます。

アプリケーションを終了 
Application.Quit();

以上を組み合わせて、例えば ESC キーを押したらUnityエディタでもアプリケーションでもゲームを終了するには Update メソッドの中で次の様に書きます。

ESCキーを押したらゲーム終了 
using の所に

#if UNITY_EDITOR 
using UnityEditor; 
#endif

を追加してから

if (Input.GetKey (KeyCode.Escape)) {

       #if UNITY_EDITOR 
       EditorApplication.isPlaying = false;
       #endif

       Application.Quit();
}
