次にゲームオブジェクトを動かしてみたいと思います。

ゲームオブジェクトを動かす方法は色々ありますが、今回は物理演算エンジンを使って動かしてみたいと思います。
そのためにはまず、以下の手順で「Rigidbody 2D コンポーネント」をゲームオブジェクトにアタッチします(※)。

※ コンポーネント(Component)は各オブジェクトの振る舞いを定めるメソッドやプロパティ(＝パラメータ)のグループのことです。
コンポーネントをゲームオブジェクトにくっつけることをアタッチと言います。
例えば Transform コンポーネントはゲームオブジェクトの位置やサイズ、向きを定めるコンポーネントで、Sprite Renderer コンポーネントは画像を描画するコンポーネントです。
Rigidbody 2D コンポーネントはゲームオブジェクトの速度や重さ、重力などを定めるコンポーネントです。

Rigidbody 2D コンポーネントのアタッチ方法:

Hierarchy ウィンドウで対象のゲームオブジェクトを選択する。
Inspector ウィンドウの Add Component ボタンを押してメニューを表示する。
Physics 2D → Rigidbody 2D を選択すると Inspector ウィンドウに Rigidbody 2D の項目が現れる。
重力が必要ない場合は Rigidbody 2D の Gravity Scale プロパティの値を 0 にする。
他のゲームオブジェクトと衝突した時に回転させたくない場合は、Constraints → Freeze Rotation → Z プロパティをチェックする。

ただし Rigidbody 2D をアタッチしただけでは任意の方向にオブジェクトを動かす事は出来ませんので、次は「Script コンポーネント」もアタッチします。

なおScript コンポーネントの実体は C# 又は JavaScript で書かれたスクリプトファイルです。
今回は C# の方を使います。

Script コンポーネントのアタッチ方法:

Assets フォルダで右クリックし、Create → C# Script を選ぶと C# のスクリプトファイルが新規作成されるので、適切なファイル名に変更する。
Hierarchy ウィンドウでスクリプトをアタッチしたいオブジェクトを選択する。
Assets フォルダの中にある、先ほど作成したスクリプトファイルを Inspector ウィンドウの上に DnD するとスクリプトがアタッチされる。
Assets フォルダの中のスクリプトファイルをダブルクリックするとエディタが起動する。
スクリプト内に Start、Update というメソッドが含まれている事を確認する(※)。

※ Start メソッドはゲームオブジェクトが表示される前に一度だけ呼び出される初期化用メソッドで、Update は描画フレームごとに繰り返し呼び出されるメソッドです。

では準備が済んだので実際にゲームオブジェクトを動かしてみましょう。

ゲームオブジェクトを動かすためには Rigidbody 2D コンポーネント の velocity (ベロシティ) プロパティに値をセットします。
この velocity は日本語では「速度」を意味する 2 次元ベクトルで、更に「velocity.x」と「velocity.y」というプロパティに分かれます。
velocity.x は横方向の速度、velocity.y は縦方向の速度であり、それらに指定した方向と速さでゲームオブジェクトが動きます(図1)。

図1. Rigidbody 2D の velociy で指定した方角に指定した速さでゲームオブジェクトが動く

ところがこの velocity の値は Inspector ウィンドウから変更出来ません。
従って以下のソース 1 に示したコードをスクリプト内の Start または Update メソッドの中に書いて velocity の値をセットする必要があります。

ソース 1: ゲームオブジェクトを移動させるスクリプト 
var x = 横方向の移動量;
var y = 縦方向の移動量;
GetComponent<Rigidbody2D>().velocity = new Vector2( x, y );

なお、障害物にぶつかったり、摩擦が無い限りは一度 velocity を設定すればその方向に動き続けます。
また重力が 0 で無い時は重力の影響も受けます。

例えばあるオブジェクトを(重力が 0 なら) 斜め右上 45 度の方向に動かしたい場合はソース 2 の様に Start または Update メソッド内に書きます。

ソース 2: ゲームオブジェクトを斜め右上 45 度の方向に動かす 
var x = 1.0f;  
var y = 1.0f;
GetComponent<Rigidbody2D>().velocity = new Vector2( x, y );

さてコーディングが済んだら以下の手順でプログラムを実行してオブジェクトを動かしてみましょう。

プログラム実行方法:

スクリプトを上書き保存して Unity エディタの画面に戻ると自動でコンパイルされる。エラーが出たら修正する。
上部にあるのうち一番左の再生ボタンを押すと GAME ビューに画面が切り替わってプログラムが実行される。
もう一度再生ボタンを押すと停止する。
