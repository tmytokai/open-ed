Unity Hub を起動すると表示される「Porjects」画面で「NEW」ボタンを押すと新規作成ウィザードが開いてプロジェクトを新規作成することが出来ます。
このウィザードで名前や保存フォルダを指定してプロジェクトを新規作成すると 図1 のようなウィンドウが表示されます。

このウィンドウのことを「Unity エディタ」と呼びます。

画面左上のエリアのことを「ヒエラルキー(Hierarchy)ウィンドウ」

中央上の空が写っているエリアのことを「ゲーム(Game)ビュー」

右側のエリアの事を「インスペクター(Inspector)ビュー」

下側のエリアの事を「プロジェクト(Project)ブラウザー」

Project ブラウザの右側パネルの事を「アセット(Assets)フォルダ」

と言います。
他にも重要なエリアがありますが、必要になったらその都度説明します。
