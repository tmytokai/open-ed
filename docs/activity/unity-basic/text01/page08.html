ゲームはタイトル画面、キャラクター選択画面、プレイ画面、ゲームオーバー画面など様々なシーン(場面)によって構成されます。
Unity ではそれぞれのシーンを別々に作り、シーンを適宜切り替えてゲームを進行します。

さてプロジェクトを新規作成した時点ではシーン(サンプルシーンと言います)は一つしかありません。
これまでの演習で大分画面がごちゃごちゃして来たので新しいシーンを作成してリフレッシュすることにしましょう。

まずアセットフォルダの中の Scenes フォルダの中を開いて見て下さい。
SampleScene というアセットが一つ入っていますがこれがサンプルシーンです。
次にアセットフォルダの空いているスペースで右クリックし、Create → Scene を選ぶと「New Scene」という新しいシーンが出来るので、名前を「Scene2」に変更して下さい。

あとは作成した Scene2 をダブルクリックすると新しいシーンが開いて、今まで作成したゲームオブジェクトが画面から消えます。
もちろん「SampleScene」をダブルクリックすれば何時でもサンプルシーンに戻れます。

なおシーンを切り替える際にゲームオブジェクトに行なった変更を保存するか聞いてくる時がありますので、保存したい場合は「Save」、保存したくない場合は「Don't Save」を押して下さい。

さて折角シーンを新規作成したので、気分転換も兼ねて背景色も変えてみましょう。
デフォルトでは空が表示されていますが、ヒエラルキーの Main Camera をクリックし、インスペクタの Camera → Clear Flags を Solid Color にすると空が消えて単色の背景になります。
次に Background をクリックすると色選択ウィンドウが出ますので真っ黒(RGBA を 0、0、0、0)にして下さい。

なおシーンビューでは空は表示されたままになります。
空の画像を変更したい場合は skybox という種類のアセットを変更する必要がありますが、自分で skybox を用意するのは面倒なので今回は説明を省略します。
