1. シーンビューの説明

ゲームビューの上にある「Scene」タブ(図1)をクリックすると「シーン(Scene)ビュー」が表示されます(図2)。


図1. Scene タブ(黄色の円で囲んでいるタブ)


図2. シーンビュー



今まで使っていたゲームビューはユーザーがプレイ中に見る画面です。
一方シーンビューはクリエイターがゲームを作成する際に補助的に利用する画面です。

さてシーンビューの右上にある風見鶏の様なアイコンを「シーンギズモ(Gizmo)」と呼びます(図3)。


図3. シーンギズモ



シーンギズモには x, y, z の文字と色付きの円錐が 3 組表示されていますが、それらは x、y、z 軸の方向を表しています。

さてシーンビューは Unity エディタの左上にある「操作ツール」を使って操作します(図4)。


図4. 操作ツール

操作ツールのアイコンは左から

  • ビュー(View)ツール
  • 移動(Move)ツール
  • 回転(Rotate)ツール
  • 拡大縮小(Scale)ツール
  • レクト(Rect)ツール
  • トランスフォーム(Transform)ツール
  • カスタム(Custom)ツール

となります。
今回はこの中でよく使われる「ビューツール」、「移動ツール」、「回転ツール」、「拡大縮小ツール」について説明します。


2. 画面の平行移動

操作ツールの中から「ビューツール」をクリックしてからシーンビュー内をドラッグすると画面が並行移動します。
またはマウスの真ん中ボタンを使ってドラッグしても平行移動します。


3. ズームイン・ズームアウト

マウスのホイールを回して下さい
すると画面がズームイン・ズームアウトします。


4. 回転

マウスの右ボタンを使ってシーンビュー内をドラッグすると画面が回転します。
またはキーボードの Alt キー、Mac なら Option キーを押しながらドラッグしても回転します。

画面を回転するとシーンギズモも回転するので現在のx、y、z軸の方向も分かります。

ちなみにシーンギズモの円錐部分をクリックして画面を回転させることも可能です。
例えば x と書いてある円錐をクリックすると x 軸が手前側に向くように画面が回転します。


5. ゲームオブジェクトの拡大表示

これまでやってきた様にヒエラルキーでゲームオブジェクトの名前をクリックするとそのゲームオブジェクトが選択されます。
さらに名前をダブルクリックするとそのゲームオブジェクトが拡大表示されます。
試しに「HAKO」などをダブルクリックしてどうなるか確認して下さい。


6. ゲームオブジェクトの並行移動

ゲームオブジェクトを選択した状態で操作ツールの中の「移動ツール」をクリックするとゲームオブジェクトに矢印が 3 本表示されます(図5)。
この矢印をドラッグするとゲームオブジェクトを平行移動出来ます。
試しに「HAKO」などを平行移動してみてください。


図5. ゲームオブジェクトの並行移動



7. ゲームオブジェクトの向きの確認

図5 の矢印はゲームオブジェクトが向いている方向を示していて、それぞれ

  • 青 : 前方向 (forward)
  • 赤 : 右方向 (right)
  • 緑 : 上方向 (up)

を表します。
この向きはキーボード操作やゲームパッド操作でゲームオブジェクトを動かす時に大事になりますので覚えておいて下さい。


8. ゲームオブジェクトの回転

ゲームオブジェクトを選択した状態で操作ツールの中の「回転ツール」をクリックするとゲームオブジェクトに球が表示されます。
この球をドラッグするとゲームオブジェクトを回転出来ます。
試しに「HAKO」などを回転してみてください。


9. ゲームオブジェクトの拡大縮小

ゲームオブジェクトを選択した状態で操作ツールの中の「拡大縮小ツール」をクリックするとゲームオブジェクトに図5の様な線が表示されます。
ただし今回は線の先が矢印ではなくて箱になっています。
この箱をドラッグするとゲームオブジェクトを拡大縮小出来ます。
試しに「HAKO」などを拡大縮小してみてください。


10. ゲームビューの視点変更

シーンビューのタブの隣りにある「Game」タブをクリックしてゲームビューに表示を戻して下さい。
すると色々視点変更したのにゲームビューの視点は以前と変わっていないことが分かります。

上で説明した様にシーンビューはゲームを作成する際に補助的に利用する画面であってプレイ中に見る画面ではないので、もしゲームビューの視点を変更したい場合はカメラの位置や向きを変える必要があります。

このカメラはヒエラルキーの中にある「Main Camera」です。
なおカメラもゲームオブジェクトの一つです。

では試しにゲームビューを表示した状態のままでヒエラルキーの「Main Camera」をクリックし、インスペクターに表示されるカメラの Position、Rotation を変えるとどうなるか確認して下さい。
またインスペクターの下にある Field of View のスライダを前後させると画面がズームイン・アウトしますのでこちらも試して下さい。
