Unity で文字列やボタンといったユーザーインターフェイス(UI)オブジェクトを画面に表示するには uGUI という機能を使います。
ここでは文字列を表示してみましょう。

まず以下の様にして描画領域の Canvas 型オブジェクトを作成し、Canvas 型オブジェクトの上に文字列を表示する Text 型オブジェクトを配置します。

Canvas 型オブジェクトと Text 型オブジェクトの新規作成

Hierarchy ウィンドウ上で右クリック → UI → Text を選択すると 「Canvas」という名前の Canvas 型オブジェクトと 「Text」という名前の Text 型オブジェクトが出来る(名前は適宜変更可能)
Canvas オブジェクトをクリックすると Inspector ウィンドウに情報が表示されるので、Canvas コンポーネントの Render Mode プロパティが「Screen Space - Overlay」になっている事を確認する(※)。

※ UI オブジェクトの描画方法を決めるプロパティです。他の設定については自分で調べて下さい。

Canvas オブジェクトをダブルクリックすると Scene ビューがズームアウトして UI オブジェクトの配置状態を確認できるので、この画面でまず Text オブジェクトの位置を調整する。
Hierarchy ウィンドウで Text オブジェクトをクリックすると Inspector ウィンドウに情報が表示される。Text コンポーネントの Text プロパティが表示される文字列である。
Text コンポーネントには他にも Font 名やサイズ、色の設定があるので適宜変更する。
Game ビューに切り替えると実際のゲーム画面が表示されるので位置やサイズなどを確認する。
Scene ビューに戻って Hierarchy ウィンドウ上の適当なキャラクターオブジェクトをダブルクリックすると Scene ビューがズームインする(※)。

※ この様に uGUI の設定方法は結構独特なので、初めて uGUI を使う人は操作に少し戸惑うかもしれません。

Text オブジェクトを追加で設置したい場合は Hierarchy ウィンドウ の Canvas オブジェクトを右クリック → UI → Text を選択する。

さてスクリプト内でテキストの文字列を変更するには次の様にします。 

using の中に

using UnityEngine.UI;

を追加してから

GameObject.Find ("Canvas 型オブジェクト名/Text 型オブジェクト名").GetComponent<Text> ().text = "文字列";

例えば MyCanvas という名前のオブジェクトの中にある Text1 という名前のオブジェクトの文字列を "HOGE" に変更したい時は次のようにします。

ソース 1: Text1 オブジェクトの文字列を変更 
using の中に

using UnityEngine.UI;

を追加してから

GameObject.Find ("MyCanvas/Text1").GetComponent<Text> ().text = "HOGE";
