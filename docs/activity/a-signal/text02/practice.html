演習 2-1 (個人PC、10分): 表計算ソフトで $f(t) = \sin( 2\pi \cdot t) + \alpha$ のグラフを描いて「上下への平行移動」を確認してみましょう。ここで $\alpha$ は $\{ 0, -1, 2 \}$ の 3 パターンとします。

表計算ソフトを起動します。
時刻の範囲は $0 \leq t \leq 2$ 秒区間、代表点は 0.01 秒刻みとします。
最初の列に各時刻をオートフィルを使って入力します。
次の列に $\alpha = 0$ とした時の $f(t)$ の値をオートフィルを使って入力します。
その次の列に $\alpha = -1$ とした時の $f(t)$ の値をオートフィルを使って入力します。
その次の列に $\alpha = 2$ とした時の $f(t)$ の値をオートフィルを使って入力します。
3つのグラフを 1 枚のグラフ領域に表します。グラフの種類は散布図、線のみ、平滑化なしとします。
保存します。ファイル名は「演習番号-チーム番号」(英数字は半角)として下さい。
チーム内で答え合わせをし、提出用ファイルに「チーム番号」と「メンバー全員の氏名」を書き入れて下さい。
クラウド上の共有フォルダに提出用ファイルをアップロードして下さい。
