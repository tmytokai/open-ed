3. 時間領域アナログ信号のグラフの描き方


演習 1-3 (個人ノート、15分): 手描きで時間領域アナログ信号 $f(t) = -t^2+2t$ のグラフを描いてみましょう。

ノートに「演習番号」を記入し番号の下に赤線を引いて下さい。
時刻の範囲は $-2 \leq t \leq 3$ 秒区間、代表点は 1 秒刻みとします。
縦軸・横軸を描きます。
各代表点を描きます。
各代表点を滑らかな線で繋ぎます。

演習 1-6 (個人PC、10分): 表計算ソフトで時間領域アナログ信号 $f(t) = -t^2+2t$ のグラフを描いてみましょう。

表計算ソフトを起動します。
時刻の範囲は $-2 \leq t \leq 3$ 秒区間、代表点は 0.01 秒刻みとします。
最初の列に各時刻をオートフィルを使って入力します。
次の列に各時刻における $f(t)$ の値をオートフィルを使って入力します。
グラフを挿入します。グラフの種類は散布図、線のみ、平滑化なしとします。
保存します。ファイル名は「演習番号-チーム番号」(英数字は半角)として下さい。
チーム内で答え合わせをし、提出用ファイルに「チーム番号」と「メンバー全員の氏名」を書き入れて下さい。
クラウド上の共有フォルダに提出用ファイルをアップロードして下さい。
