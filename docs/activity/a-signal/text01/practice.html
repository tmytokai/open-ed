3. 時間領域アナログ信号のグラフの描き方


演習 1-3 (個人、20分): 任意のドローソフトを使い、手描きで時間領域アナログ信号

$f(t) = -t^2+2t$

のグラフを描いてみましょう。

ドローソフトを起動します。
縦軸・横軸を描きます。時刻の範囲は $-2 \leq t \leq 3$ 秒区間、代表点は 1 秒刻みとします。
各代表点を描きます。
各代表点を滑らかな線で繋ぎます。
画像を保存します。ファイル名は「AS-1-3」として下さい(画像形式は任意)。
チーム内で答え合わせをし、提出用の画像ファイルを選んで下さい。
指定場所に「画像ファイル」を貼り付けて下さい。

演習 1-6 (個人、20分): 任意の表計算ソフトで時間領域アナログ信号

$f(t) = -t^2+2t$

のグラフを描いてみましょう。

表計算ソフトを起動します。
時刻の範囲は $-2 \leq t \leq 3$ 秒区間、代表点は 0.01 秒刻みとします。
最初の列に各時刻(秒)をオートフィルを使って入力します。
次の列に各時刻における $f(t)$ の値をオートフィルを使って入力します。
グラフを挿入します。グラフの種類は散布図、線のみ、平滑化なしとします。
シートを保存します。ファイル名は「AS-1-6」として下さい。
チーム内で答え合わせをし、提出用のグラフを画像ファイルとして保存して下さい。
指定場所に「画像ファイル」を貼り付けて下さい。
