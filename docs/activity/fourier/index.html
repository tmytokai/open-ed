目的

あなたが周期性時間領域アナログ信号に対してフーリエ級数展開を行える技術者になること

必要性

フーリエ級数展開によって周期性時間領域アナログ信号を正弦波の和に分解できます。
この正弦波の振幅や位相の事をスペクトルと言います。
従ってスペクトル解析を学ぶために、技術者はフーリエ級数展開を学ぶ必要があります。
