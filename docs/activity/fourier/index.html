所属コース: アナログ信号のスペクトル解析

上位ナレッジ(ナレッジマップ):

  1. アナログサイン波
  2. 複素数と複素平面
  3. 複素正弦波

最終更新日: 2020/7/6

目的

あなたが周期性時間領域アナログ信号に対して複素フーリエ級数展開を行える技術者になること

必要性

複素フーリエ級数展開によって周期性時間領域アナログ信号を複素正弦波の和に分解できます。
この複素正弦波の絶対値や偏角の事をスペクトルと言います。
従ってスペクトル解析を学ぶために、技術者は複素フーリエ級数展開を学ぶ必要があります。

進め方

もし上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
またこのアクティビティの具体的な進め方についてはこちらを参照して下さい。
