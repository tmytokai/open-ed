\noindent \textbf{【出題意図】}

\noindent 基本波の周波数を求めることが出来るか確かめる問題である。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【重要事項】}

\noindent フーリエの定理

\smallskip
$f(t)$ が周期 $\textrm{T}$ [秒] の周期性時間領域アナログ信号ならば、$f(t)$ を

\smallskip
1. 直流成分

\smallskip
2. 基本周波数 $f_1$ [Hz]を持つ時間領域アナログサイン波 (基本波と呼ぶ)

\smallskip
3. 基本周波数 $f_1$ [Hz]の正整数倍の周波数を持つ無限個の時間領域アナログサイン波 (高調波と呼ぶ)

\smallskip
の和に分解できる。


%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【解説】}

\noindent 周期が $\textrm{T}=1$ であるので、基本波の周波数は基本周波数 $f_1 = 1$ [Hz] となる。
