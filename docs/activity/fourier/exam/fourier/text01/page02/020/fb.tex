\noindent \textbf{【出題意図】}

\noindent 周期性時間領域アナログ信号の周期から基本波や高調波の角周波数を求めることが出来るか確かめる問題である。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【重要事項】}

\noindent フーリエの定理

\smallskip
$f(t)$ が周期 $\textrm{T}$ [秒] の周期性時間領域アナログ信号ならば、$f(t)$ を

\smallskip
1. 直流成分

\smallskip
2. 基本周波数 $f_1$ [Hz]を持つ時間領域アナログサイン波 (基本波と呼ぶ)

\smallskip
3. 基本周波数 $f_1$ [Hz]の正整数倍の周波数を持つ無限個の時間領域アナログサイン波 (高調波と呼ぶ)

\smallskip
の和に分解できる。


%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【解説】}

\noindent 周期 $\textrm{T}=1$ より基本角周波数は $w_1 = 2\pi$ [rad/秒] なので第2高調波の角周波数は $w = 2\cdot w_1 = 4\pi$ [rad/秒]となる。
