オーバーロード


「オーバーロード(overload)」は日本語では「多重定義」と呼ばれていますが、一般的にはオーバーロードと呼ばれていますので、このアクティビティでもオーバーロードに呼び方を統一します。

またオーバーロードには主に「メソッドのオーバーロード」と「演算子のオーバーロード」がありますが、今回は「メソッドのオーバーロード」だけ取り扱います。 ただし「演算子のオーバーロード」の方も大変良く使われる機能ですので、もし今後必要になったら自分で調べて下さい。

さて「メソッドのオーバーロード」とは簡単に言うと

名前は同じだけど引数の個数や型が違うメソッドを複数定義する機能

の事です。

C言語の様に(メソッドの)オーバーロードが無い言語が無い言語の場合は、 ほとんど同じ様な機能を持っているメソッドなのに、引数の数や型が少し違うだけで名前を変える必要があるため、hoge1()とかhoge2()とかhoge3()とか似たような名前のメソッドが氾濫し、プログラマーも「この型の場合はどのメソッド名だっけ？」みたいにいちいち気にする必要が出てきます。

一方、Javaの様にオーバーロードがある言語の場合は引数の数や型が違ってもメソッドの名前を変える必要がないので、メソッド名の氾濫を防いでソースコードがすっきりクールになりますし、プログラマーもメソッド名をいちいち気にする必要が無くなるというメリットがあります。

特に重要で一番使用例が多いオーバーロードはコンストラクタのオーバーロードです。 コンストラクタのオーバーロードについては実際に演習で試してみたいと思います。

では実際にオーバーロードの例を見てみましょう。 まず以下の OLtest というクラスを定義します。 
ソース1:

public class OLtest {

    public void show(){
    	System.out.println("引数なし");
    }
    
    public void show( String a){
    	System.out.println("引数は文字列: " + a);
    }

    public void show( int a){
    	System.out.println("引数は整数: " +a);
    }

    public void show( int a, int b){
    	System.out.println("引数は整数x2: " +a + "-" + b);
    }
}

次に この main メソッドを実行します。 
ソース2:

public static void main(String[] args) {

   OLtest ol = new OLtest();
   ol.show();
   ol.show( "ほげ" );
   ol.show( 123 );
   ol.show( 123, 456 );
}

すると次の実行結果になります。 
ソース2の実行結果:

引数なし
引数は文字列: ほげ
引数は整数: 123
引数は整数x2: 123-456

このように、同じ名前のメソッドを呼び出しているのに、引数の数や型が違うと異なる表示がされます。

一方、もしJavaにオーバーロードが無かったらソース3のようにクラスを書かないと駄目になります。
もちろんこのクラスでも動きますが、ソース4のように引数の数や型毎にいちいち名前を変えないといけないので野暮ったくなります。
また「引数が文字型なのはshow2()だっけ、show3()だっけ？」とプログラマーも悩む必要が出てきます。 
ソース3: もしオーバーロードが無かったら・・・

public class OLtest {

    public void show1(){
    	System.out.println("引数なし");
    }
    
    public void show2( String a){
    	System.out.println("引数は文字列: " + a);
    }

    public void show3( int a){
    	System.out.println("引数は整数: " +a);
    }

    public void show4( int a, int b){
    	System.out.println("引数は整数x2: " +a + "-" + b);
    }
}


ソース4: 野暮ったい

public static void main(String[] args) {

   OLtest ol = new OLtest();
   ol.show1();
   ol.show2( "ほげ" );
   ol.show3( 123 );
   ol.show4( 123, 456 );
}
