複数インスタンスの生成

クラスはC言語で言う「構造体」、インスタンスは「mallocで確保した構造体のメモリ領域」に相当しますので、構造体の様にクラスをひとつ定義しておけば、それを元にして複数のインスタンスを生成することが出来ます。

例えば次のソースでは Score クラスの定義を元に太郎の得点(sc_taro)と花子の得点(sc_hanako)の２つのインスタンスを new 演算子で作っています。 
ソース1:

public static void main(String[] args) {
	Score sc_taro = new Score();
	sc_taro.math = 5;
	sc_taro.english = 3;
	
	Score sc_hanako = new Score();
	sc_hanako.math = 80;
	sc_hanako.english = 70;
	
	System.out.println( "太郎: math = " + sc_taro.math+ " english = " + sc_taro.english );
	System.out.println( "花子: math = " + sc_hanako.math+ " english = " + sc_hanako.english );
}
