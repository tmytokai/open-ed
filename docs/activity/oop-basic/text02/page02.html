標準例外


前のページのソース1に出てきた FileNotFoundException クラスは Java に初めから備わっている例外クラスでしたが、これ以外にも Java には例外(つまりエラー)の種類に合わせて様々な例外クラスが用意されています。

これらの、 Java に初めから備わっている例外クラス(あるいは例外処理そのもの)の事を「標準例外」と言います。

この「標準例外」は別に Java だけの機能ではなくて、オプジェクト指向プログラミング言語には大抵備わっています。
標準例外の種類は言語によって変わりますが、以下の標準例外は大抵の言語に備わっています(もちろん他にも沢山あります)。

・ファイル・ノットファウンド例外: ファイルが見つからない

・ゼロ除算例外: ゼロで数字を割ろうとした

・NULL ポインター例外: NULL ポインターにアクセスしようとした


Java では上の 3 つの標準例外は次の例外クラスに割当たっています。

・ファイル・ノットファウンド例外 → FileNotFoundException

・ゼロ除算例外 → ArithmeticException

・NULLポインター例外 → NullPointerException
