  2. 学習項目: [2] 例外処理

例外クラスを自作して「throw」を使って投げる事も出来ます。
自作例外クラスの作り方はプログラミング言語によって変わりますが、Java では次のように作ります。

ソース1: 
public class MyException extends RuntimeException {

	  MyException(){
		  super("自作した例外です");
	  }	 
}

ここで出てきた「extends RuntimeException」とか「super」は「継承」のテキストで説明しますので、今の所はこう書くんだ程度の認識でいて下さい。
またこの自作例外クラスの作り方はかなり簡略化したものですので、実際の業務等で使う場合はちゃんとしたやり方を調べてから作って下さい。

後は実行中に何か問題が生じたら、次のように throw 文でこの自作例外をnewして投げて catch で受け取るだけです。
「throw new MyException();」が自作例外を投げて(throw)している部分です。

ソース2: 
     try{ 
         if( 何か問題があったら ) throw new MyException();
     } 
     catch( MyException e ) { 
         System.out.println( e ); 
     }

これを実行して何か問題が生じたら「MyException: 自作した例外です」と表示されます。
