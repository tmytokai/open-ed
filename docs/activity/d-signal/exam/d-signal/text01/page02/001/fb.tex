\noindent \textbf{【出題意図】}

\noindent 時間領域ディジタル信号の定義を理解しているかどうかを確かめる問題である。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【重要事項】}

\medskip
\noindent 時間領域ディジタル信号とは独立変数 $i$ が時刻を表す信号のことである。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【解説】}

\noindent 非音声データや動きの無い画像データは時間変化が無いので時間領域ディジタル信号ではない。
