\noindent \textbf{【出題意図】}

\noindent 定義域と値域を理解しているかどうかを確かめる問題である。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【重要事項】}

\bigskip
\noindent 信号 $f[i]$ の独立変数 $i$ が取り得る値の範囲の事を定義域という。

\noindent 一方 $f[i]$ そのものが取り得る値の範囲のことを値域という。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【解説】}

\noindent 重要事項より求まる。
