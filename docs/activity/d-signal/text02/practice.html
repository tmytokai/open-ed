1. サンプリング(標本化)


演習 2-1 (チーム、10分): 皆さんが常日頃から利用している CD、ハイレゾ、DSD(Direct Stream Digital)、VoLTE(AMR-WB) のサンプリング周波数 $f_s$ [Hz]を調べましょう。

ホワイトボードに「演習番号」「チーム番号」「メンバー全員の氏名」を記入して下さい。
チームみんなで手分けして Web 上等で調べ、各周波数をホワイトボードに書いて下さい。
ホワイトボードの写真を撮り、クラウド上の共有フォルダに写真をアップロードして下さい。


演習 2-2 (チーム、10分): サンプリング周波数 $f_s$ [Hz]を大きくした時のメリットとデメリットを調べてみましょう。
※ $f_s$ を小さくしたときはメリットとデメリットが逆になりますのでメリットだけ挙げてください。

ホワイトボードに「演習番号」「チーム番号」「メンバー全員の氏名」を記入して下さい。
チームみんなで手分けして Web 上等で調べ、メリットを1人1件ホワイトボードに書いて下さい。
ホワイトボードの写真を撮り、クラウド上の共有フォルダに写真をアップロードして下さい。


演習 2-3 (個人PC、15分): 表計算ソフトで時間領域アナログ信号 $f(t) = \sin( 2\pi\cdot 2 \cdot t ) + \sin( 2\pi\cdot 4 \cdot t )$ をサンプリング周波数 $f_s = 10$ [Hz]で $\textrm{T}_s = 4$ 秒間サンプリングして得られる時間領域ディジタル信号 $f[i]$ のグラフを描いてみましょう。

表計算ソフトを起動します。
時刻 $i$ の範囲を求めてください。
最初の列に各 $i$ をオートフィルを使って入力します。
次の列に各 $i$ における $f[i]$ の値をオートフィルを使って入力します。
グラフを挿入します。グラフの種類は散布図、点あり、線あり、平滑化ありとします。
保存します。ファイル名は「演習番号-チーム番号」(英数字は半角)として下さい。
チーム内で答え合わせをし、提出用ファイルに「チーム番号」と「メンバー全員の氏名」を書き入れて下さい。
クラウド上の共有フォルダに提出用ファイルをアップロードして下さい。
