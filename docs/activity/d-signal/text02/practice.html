1. サンプリング(標本化)

演習 2-1 (チーム、10分): 皆さんが常日頃から利用している CD、ハイレゾ、DSD(Direct Stream Digital)、VoLTE(AMR-WB) のサンプリング周波数 $f_s$ [Hz]を調べましょう。

指定場所に「メンバー全員の氏名」を記入して下さい。
チームみんなで手分けして Web 上等で調べ、指定場所に「各サンプリング周波数」を書いて下さい。


演習 2-2 (チーム、10分): サンプリング周波数 $f_s$ [Hz]を大きくした時のメリットとデメリットを調べてみましょう。

指定場所に「メンバー全員の氏名」を記入して下さい。
チームみんなで手分けして Web 上等で調べ、指定場所に「メリットとデメリット」をそれぞれ2件以上書いて下さい。


演習 2-3 (個人PC、15分): 表計算ソフトで時間領域アナログ信号

$f(t) = \sin( 2\pi\cdot 2 \cdot t ) + \sin( 2\pi\cdot 4 \cdot t )$

をサンプリング周波数 $f_s = 10$ [Hz]で 0 秒地点から $\textrm{T}_s = 4$ 秒間サンプリングして得られる時間領域ディジタル信号 $f[i]$ のグラフを描いてみましょう。

表計算ソフトを起動します。
時刻 $i$ の範囲を求めてください。
最初の列に各 $i$ をオートフィルを使って入力します。
次の列に各 $i$ における $f[i]$ の値をオートフィルを使って入力します。
グラフを挿入します。グラフの種類は散布図、点あり、線あり、平滑化ありとします。
シートを保存します。ファイル名は「DS-2-3」として下さい。
チーム内で答え合わせをし、提出用のグラフを画像ファイルとして保存して下さい。
指定場所に「メンバー全員の氏名」を記入して下さい。
指定場所に「画像ファイル」を貼り付けて下さい。
