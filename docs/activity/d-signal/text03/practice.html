演習 3-3 (個人、10分):

時間領域アナログ信号(サイン波)

\[ f(t) = \sin( 2 \pi t ) \]

を、サンプリング周波数 $f_s = 4$ [Hz]、量子化ビット数 $q = 1$ [bit]で $t=0$ 秒時点から 1.75 秒間サンプリングした時のディジタルデータを 16 進数(0x??)で求め、各自が指定した場所に記入して下さい。
なお、$f(t)$ の値が 0 より小さい時は 0b0、$f(t)$ の値が 0 以上の時は 0b1 で量子化することにします。
