演習4-1 (個人、? 分): 4次ARモデル( a[1] = 0.1、a[2] = 0.3、a[3] = 0.2、a[4] = 0.15 )

\[ x[i] = -0.1 \cdot x[i-1] -0.3 \cdot x[i-2] -0.2 \cdot x[i-3] - 0.15 \cdot x[i-4] + err[i] \ , \ (i=0,1,\cdots, \textrm{N}-1) \]

から出力されたディジタル信号 x[i] に対してレビンソン・ダービン・アルゴリズムを適用し、各 LPC 係数を推定してみましょう。
信号の長さは N = 50000 とします。
また x[-1] = x[-2] = x[-3] = x[-4] = 0 とします。
また $err[i]$ は平均 0、分散 1 の独立な正規乱数とします。

テンプレートソース(D-FILTER-4-1.cpp)をダウンロードします。
テンプレートソースを開いて穴埋めをします。
コンパイルして実行すると L 個の LPC 係数の推定値とPARCOR係数が画面上に表示されます。
さらに 1 つのCSVファイル(D-FILTER-4-1-x.csv)が出力されます。D-FILTER-4-1-x.csv は x[i] の先頭 100 データです。
D-FILTER-4-1-x.csv を表計算ソフトで開き、一行目に空行を挿入し、最初の列の先頭に i 、次の列の先頭に x[i] と入力します。
グラフを挿入します。グラフの種類は散布図、点あり、線あり、平滑化「なし」とします。
「名前を付けて保存」でシートを表計算形式で保存します。ファイル名は「D-FILTER-4-1-x」として下さい(拡張子は使用する表計算ソフトによる)。
チーム内で答え合わせをして下さい。
各自が指定場所に「LPC係数の推定値」「PARCOR係数の値」「x[i] のグラフ」を貼り付けて下さい。

演習4-4 (個人、? 分): 自分の声の LPC 係数の推定値を求め、さらにその推定値を使った時の線形予測値 $\hat{x}[i]$ と線形予測誤差 $err[i]$ を求めてみましょう。なお L = 100 とします。

マイクを PC の端子に繋ぎます。
マイクの設定を開いて音量調整をします。
レコーダーを開きます。
「アー」の声を「2 秒以上」録音し、ダウンロードフォルダに音声ファイル( out.wav ) をダウンロードします。
out.wav を試しに再生し、もし音が小さい場合は録音し直します。
out.wav を演習フォルダに移動して「 a.wav 」に名前を変更します。
テンプレートソース(D-FILTER-4-4.cpp)をダウンロードします。
テンプレートソースを開いて穴埋めをします。
コンパイルして実行すると1 秒地点から 0.5 秒分の音声を切り出した「 D-FILTER-4-4-out.wav 」、およびLPC係数の推定値を使った時の線形予測値 「 D-FILTER-4-4-hatx.wav 」、線形予測誤差「D-FILTER-4-4-err.wav」が出来ます。
線形予測誤差「D-FILTER-4-4-err.wav」のスペクトログラム画像を表示してください。
チーム内で答え合わせをして下さい。 また出力された各 Wave ファイルを聴き比べて下さい。
各自が指定場所に「ソースコード(※ 「ここから」〜「ここまで」の部分だけ)」と「D-FILTER-4-4-err.wavのスペクトログラム画像」を貼り付けて下さい。
