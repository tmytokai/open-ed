2. 移動平均フィルタ


演習2-1 (個人): 移動平均フィルタを使ってサンプル音声から高周波成分を取り除いてみましょう。

テンプレートソース(D-FILTER-2-1.cpp)をダウンロードします。
サンプル音声(trumpet_22k_mono.wav)をソースと同じフォルダにダウンロードします。
テンプレートファイルの記述に従って ? の部分を上から穴埋めしてプログラムを完成させて下さい。なおサンプリング周波数は $f_s = 22050$ [Hz]、フィルタ長は $\textrm{L} = 31$ とします。
コンパイルして実行すると「 D-FILTER-2-1.wav 」という Wave形式音声ファイルが出来ます。
元の音声(trumpet_22k_mono.wav)を演奏し、スペクトログラム画像を確認してください。
フィルタリング後の音声(D-FILTER-2-1.wav)を演奏し、スペクトログラム画像を見て高周波成分が減衰していることを確認して下さい。
指定場所に「ソースコード」「元の音声のスペクトログラム画像」「フィルタリング後の音声のスペクトログラム画像」を貼りつけて下さい。


演習2-2 (個人): 移動平均フィルタのインパルス応答のグラフを描いてみましょう。

パラメータ値は $\textrm{L} = 31$ とします。
表計算ソフトを起動し、一行目の最初の列の先頭に「i」、次の列の先頭に「h[i]」と入力します。
最初の列に $i = 0,1,\cdots,\textrm{L}-1$ をオートフィルを使って入力します。
次の列に各 $h[i]$ の値をオートフィルを使って入力します。
グラフを挿入します。グラフの種類は散布図、点あり、線あり、平滑化なしとします。
「名前を付けて保存」でシートを表計算形式で保存します。ファイル名は「D-FILTER-2-2」として下さい(拡張子は使用する表計算ソフトによる)。
チーム内で答え合わせをして下さい。
指定場所に「インパルス応答のグラフ」を貼り付けて下さい。
