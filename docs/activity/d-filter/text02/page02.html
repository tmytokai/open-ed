次は(単純)移動平均フィルタ(Simple Moving Average Filter)について説明します。

この移動平均フィルタは現在時刻時間を移動させながら過去 $\textrm{L}$ 時刻分の入力信号値の単純平均を出力するというだけのフィルタですので、インパルス応答は次のようになります。

(単純)移動平均フィルタのインパルス応答:

さらに移動平均フィルタフィルタ長として適当な正整数 $\textrm{L}$ を決める。

この時、移動平均フィルタのインパルス応答は次の式で表される。

\begin{align*} h[i] = \frac{1}{\textrm{L}}\ ,\ (i = 0, 1, \cdots, \textrm{L}-1 ) \end{align*}

伝達関数は次のようになります。

移動平均フィルタの伝達関数: \begin{align*} \textrm{H}(z) = \frac{1}{\textrm{L}}\cdot\sum_{i=0}^{\textrm{L}-1} \{ z^{-i} \} \end{align*}

移動平均フィルタの周波数特性はFIRローパスフィルタの時と同様に考えて次のようになります。
ただし式を簡単にするために今回は $\textrm{L}$ は奇数とし、$\textrm{C} = (\textrm{L}-1)/2$ を定義しました。

移動平均フィルタの周波数特性:

$\textrm{L}$ は奇数とし、$\textrm{C} = (\textrm{L}-1)/2$ とする。

\begin{align*} \textrm{H}(w) = \left ( \frac{1}{\textrm{L}} + \sum_{i = 1}^{\textrm{C}} \left \{ \frac{1}{\textrm{L}}\cdot 2 \cdot \cos (i\cdot w \cdot \tau ) \right \} \right ) \cdot \textrm{e}^{ -j \cdot w \cdot \textrm{C} \cdot \tau } \end{align*}

この後演習で確認しますが、移動平均フィルタの周波数特性はローパス特性になります。
ただしフィルタ長 $\textrm{L}$ が短いと緩慢な減衰特性になるので、高周波ノイズをきちんと取り除きたい場合はある程度の長さにしないといけません。
