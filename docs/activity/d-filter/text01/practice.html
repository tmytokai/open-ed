Linux 版 (template01-2.cpp)
Windows 版 (template01-2-SJIS.cpp)

テンプレートファイルの記述と下のFIRローパスフィルタ係数の求め方に従って ? の部分を上から穴埋めしてプログラムを完成させて下さい。なおデフォルト値はフィルタ次数(タップ数) $\textrm{L}$ = 63、カットオフ周波数 $f_c$ = 1000 [Hz] とします。

FIR ローパスフィルタ係数の求め方:

フィルタ次数(タップ数) $\textrm{L}$、カットオフ周波数 $f_c$ [Hz]、サンプリング周波数 $f_s$ [Hz] における FIR ローパスフィルタ係数(=インパルス応答) $h[i]$ は以下の式で得られる。なお $\textrm{C} = (\textrm{L}-1)/2$ である。

\begin{align*} \alpha &= \frac{f_c}{(f_s/2)} \\ h[\textrm{C}] &= \alpha \\ h[\textrm{C}-i] &= h[\textrm{C}+i] = \frac{ \sin \left ( \pi \cdot i \cdot \alpha \right ) }{ \pi \cdot i } \quad , \quad ( i = 1, 2,・・・, \textrm{C} ) \end{align*}
