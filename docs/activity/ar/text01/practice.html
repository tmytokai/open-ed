自己回帰モデル


演習1-1 (個人): 以下の仕様の自己回帰モデルから信号列 $x[i]$ を生成してみましょう。

仕様:

  1. 次数は L = 4
  2. 信号の長さは N = 100
  3. LPC 係数は a[1] = 0.1、a[2] = 0.3、a[3] = -0.2、a[4] = 0.15
  4. $x[-1] = x[-2] = x[-3] = x[-4] = 0$
  5. ホワイトノイズは平均 0、分散 1 の時刻毎に独立な正規乱数

フォルダ「AR_1_1」を新規作成し、その中に移動します
フォルダ「AR_1_1」の中に main.cpp を作成します
main.cpp を開き、ファイルの中身をテンプレートソースの内容に置き換えて下さい。
テンプレートファイルの記述に従って ? の部分を上から穴埋めしてプログラムを完成させて下さい。
コンパイルして実行すると 1 つの CSV ファイル (AR_1_1_x.csv)が出力されます。この CSV ファイルは信号値 $x[i]$ の N = 100 個分の値です。
CSV ファイルを表計算ソフトで開き、一行目に空行を挿入し、最初の列の先頭に「i」、次の列の先頭に「x[i]」と入力します。
グラフを挿入します。グラフの種類は散布図、点なし、線あり、平滑化なしとします。
「名前を付けて保存」でシートを表計算形式で保存します。ファイル名は「AR_1_1_x」として下さい(拡張子は使用する表計算ソフトによる)。
チーム内で答え合わせをしてください。
指定場所に「main.cpp のソースコード」と「グラフ」を貼り付けて下さい。


線形予測モデル


演習1-2 (個人): 演習 1-1 で生成した信号列 $x[i]$ の線形予測値を求めてみましょう。

フォルダ「AR_1_2」を新規作成し、その中に移動します
フォルダ「AR_1_2」の中に main.cpp を作成します
main.cpp を開き、ファイルの中身をテンプレートソースの内容に置き換えて下さい。
テンプレートファイルの記述に従って ? の部分を上から穴埋めしてプログラムを完成させて下さい。
コンパイルして実行すると 1 つの CSV ファイル(AR_1_2_hat_x.csv)が出力されます。この CSV ファイルは 線形予測値 $\hat{x}[i]$ のN = 100 個分の値です。
CSV ファイルを表計算ソフトで開き、一行目に空行を挿入し、最初の列の先頭に「i」、次の列の先頭に「hat_x[i]」と入力します。
グラフを挿入します。グラフの種類は散布図、点なし、線あり、平滑化なしとします。
縦軸の範囲を演習1-1で書いたグラフと合わせます。
「名前を付けて保存」でシートを表計算形式で保存します。ファイル名は「AR_1_2_hat_x」として下さい(拡張子は使用する表計算ソフトによる)。
チーム内で答え合わせをし、元の信号 $x[i]$ のグラフと線形予測値 $\hat{x}[i]$ のグラフを見比べてみましょう。

※ $\hat{x}[i] = x[i]-e[i]$ なので、$\hat{x}[i]$ は各時刻でホワイトノイズ $e[i]$ を取り除いた $x[i]$ の予測値とみなせます

指定場所に「main.cpp のソースコード」と「グラフ」を貼り付けて下さい。
