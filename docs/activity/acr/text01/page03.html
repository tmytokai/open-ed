  2. 学習項目: [1] 自己相関関数

$f[i]$ が一様分布や正規分布の様な白色雑音(ホワイトノイズ)の場合は $R[0]$ が最大値になるのは通常の信号の時と同じなのですが、 $n > 0$ の $R[n]$ の値は $f[i]$ と $f[i+n]$ が無相関なのでほぼ一定値になります。

具体的には $f[i]$ が平均 $\mu$、分散 $\sigma^2$ の白色雑音なら

$R[0] = \mu^2 + \sigma^2$
$R[n] = \mu^2 \ ,\ n \geq 1$

の値に近づきます。

ではここで例として平均 $\mu = -2$、分散 $\sigma^2 = 9$ の正規乱数を N=10000 点生成した信号を考えます。
実際に得られた信号は以下の図1です。

この信号の自己相関関数のグラフが以下の図2です。

この様に自己相関関数を計算してみたら $R[0]$ は $\mu^2 + \sigma^2 = 13$ に近く、その他の自己相関関数値は $\mu^2 = 4 $ に近い値でほぼ一定なので、元の信号は白色雑音である(可能性がある)事が分かりました。
