4. 自動運転車プログラミング

センサーなどのパーツを車につけただけではとうぜん車は動きません。 ハンドルやエンジンやブレーキに適切な指示をだして車を自動で動かすためには

  • 地図やGPSやセンサーなどから送られてきた情報をひとまとめにして分析(ぶんせき)する
  • 分析した結果を使ってハンドル、ブレーキ、エンジンを操作する

というプログラムをつくる必要があります。

今回はこの自動運転車プログラミングのやり方について学びます(さすがに本格的な自動運転車プログラミングをスクラッチで学ぶのはむずかしいですが、その雰囲気だけでも味わってください)。
