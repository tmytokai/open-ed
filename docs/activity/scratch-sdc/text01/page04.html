4. 自動運転車プログラミング

もちろんGPSやセンサー、カメラなどのパーツを車につけただけでは車は動きません。走行ルートを決め、ハンドルやエンジンやブレーキに適切な指示をだして車を自動で動かすためには

  • 地図やGPSなどの情報を使って目的地までの走行ルートを決める
  • センサーやカメラなどの情報を使ってハンドル、ブレーキ、エンジンを操作する

というプログラムをつくる必要があります。今回はこの自動運転車プログラミングのやり方について学びます。

さすがに本格的な自動運転車プログラミングをスクラッチで学ぶのはむずかしいですが、その雰囲気だけでも味わってください。
