表示と入力と乱数

Hello World を作ってみたので、次は文字列や変数の表示や入力、乱数の取得をしてみましょう。


(1) コンソールへの表示

文字列のコンソールへの表示は 
改行あり: System.out.println( "文字列" );
または
改行無し: System.out.print( "文字列\n" );
で行います。println と print の違いは行の最後に改行するかどうかだけです。

intやdoubleなどの数字が入っている変数の内容も同じように表示できます。
例えば main メソッドの続きに以下のコードを追加して実行してみて下さい。 


int a = 12;
System.out.println( a );
文字列と数字型変数の内容を + で繋いで同時に表示することも出来ます。
例えば上のコードの続きに以下のコードを追加して実行してみて下さい。 


System.out.println( "aの内容は" + a + "です" );
ただし、当然ですが数字型の変数と数字型の変数を + で繋ぐと2つの値が足された値が表示されますので注意して下さい。
例えば上のコードの続きに以下のコードを追加して実行すると「123.4」ではなくて「15.4」が表示されます。 


double b = 3.4;
System.out.println( a + b ); // ※1


(2) コンソールからのキーボード入力

次に Scanner というクラスを使ってコンソールからキーボードで文字列や数字を入力する方法について学びましょう。
クラスとは何か？ということに関してはおいおい説明しますので、現時点では次のようにすればキーボード入力が出来るとだけ覚えておいて下さい。

まずキーボード入力を可能にするために、ソースコードの先頭に次の文を追加して下さい。 


import java.util.Scanner;
import は C 言語の include の様なものです。
では文字列の入力をしてみましょう。とりあえず次のコードを main メソッドに追加して実行し、キーボードから何か文字を打ってみて下さい。 


Scanner scan = new Scanner(System.in);

System.out.print( "文字列: ");
String str = scan.next();
System.out.println( str );
String は文字列型のクラスですが、今の所は文字列を入れる変数のようなものだと思ってくれて結構です。
上の様に、 scan.next() によって文字列をキーボードから入力して str に代入することが出来ます。

また double 型と int 型の入力をしたい場合は次のように scan.nextDouble() と scan.nextInt() を使います。
Scanner scan で定義した scan は使い回しできますので、上のコードの続きに以下のコードを追加して実行してみて下さい。 


System.out.print( "実数: ");
double c = scan.nextDouble(); 
System.out.println( c );

System.out.print( "整数: ");
int d = scan.nextInt(); ※2
System.out.println( d );
なお数字以外の文字列を入力するとエラーが出てプログラムが終了します。なおJavaのバージョンによってはscanを使い終わったら scan.close() で scan　を閉じないと警告がでるみたいです。


(3) 乱数の取得

乱数は Math クラスを使って簡単に取得できます。

0から1までの実数の乱数を取得したい場合は次のように Math.random() を使います。
コードの続きに以下のコードを追加して実行してみて下さい。 


double e = Math.random();
System.out.println( e );
従ってAからBまでの整数の乱数を取得したい場合は次のように書きます。
コードの続きに以下のコードを追加して実行してみて下さい。 「(B-A) + 1」 と 1 を足さないとAからB-1までの乱数になるので注意して下さい。 


int A = 5;
int B = 8;
int f = A + (int)(Math.random()*((B-A)+1));
System.out.println( f );
