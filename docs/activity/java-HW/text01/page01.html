以下の手順で、eclipse 上で Java を使って Hello World を作って実行してみましょう。

端末を開き

$ eclipse &

でeclipseを起動する。
「ワークスペースの選択」ダイアログが出たらそのままOKを押す。

※「ワークスペース」とは作成するプロジェクトを保存する場所のこと

初回起動時は時間がかかるので起動するまでしばらく待つ。 起動後に「ようこそ」画面が出たら、タブの×を押して「ようこそ」画面を閉じる。
初回起動時はメニューの「ファイル」→「新規」→「プロジェクト」→「Javaプロジェクト」の順で選択する。
2回目以降はメニューの「ファイル」→「新規」→「Javaプロジェクト」の順で選択する。
「新規Javaプロジェクト」ウィザードが出たら、「プロジェクト名」に「Hello」と入れて「完了」を押す(※)。

※ プロジェクト名に日本語(全角文字)が入っているとコンパイル出来ない時があるので、プロジェクト名は半角文字だけにしてください

画面左側にある「パッケージ・エクスプローラ」の「Hello」を右クリックして「新規」→「クラス」を選ぶ。
「新規Javaクラス」ダイアログが出たら「名前」に「Main」と入れ、「どのメソッド・スタブを作成しますか？」の「public static void main(String[] args)」にチェックを入れて「完了」を押す。
もしソースコード(Main.java)が表示されなかったり、間違ってタブを閉じてしまった場合は、パッケージ・エクスプローラの三角をクリックして行って Main.java を見つけてダブルクリックする。
mainメソッド(関数)の「// TODO 自動生成されたメソッド・スタブ」の下に

System.out.println( "Hello");

をコピペする。
ツールバー上にある「実行」ボタンを押す。
「保管して起動」ダイアログが出たら「OK」を押すとソースコードが保存されて実行される。
画面下の「コンソール」に「Hello」と表示されるのを確認する。
