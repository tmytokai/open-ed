Hello World

まず以下の手順で、eclipse 上で Java を使って Hello World を作って実行してみましょう。


(1) 端末を開き

$ eclipse &

でeclipseを起動する。


(2) 「ワークスペースの選択」ダイアログが出たらそのままOKを押す。

※「ワークスペース」とはは作成するプロジェクトを保存する場所のこと


(3) 初回起動時は時間がかかるので起動するまでしばらく待つ。 起動後に「ようこそ」画面が出たら、タブの×を押して「ようこそ」画面を閉じる。


(4) メニューの「ファイル」→「新規」→「Javaプロジェクト」の順で選択する。


(5) 「新規Javaプロジェクト」ウィザードが出たら、「プロジェクト名」に「Hello」と入れて「完了」を押す(※)。

※ プロジェクト名に日本語(全角文字)が入っているとコンパイル出来ない時があるので、プロジェクト名は半角文字だけにしてください


(6) 画面左側にある「パッケージ・エクスプローラ」の「Hello」を右クリックして「新規」→「クラス」を選ぶ。


(7) 「新規Javaクラス」ダイアログが出たら「名前」に「Main」と入れ、「どのメソッド・スタブを作成しますか？」の「public static void main(String[] args)」にチェックを入れて「完了」を押す。


(8)もしソースコード(Main.java)が表示されなかったり、間違ってタブを閉じてしまった場合は、パッケージ・エクスプローラの三角をクリックして行って Main.java を見つけてダブルクリックする。


(9) mainメソッド(関数)の「// TODO 自動生成されたメソッド・スタブ」の下に「System.out.println( "Hello");」をコピペする。


(10) ツールバー上にある「実行」ボタンを押す。


(11) 「保管して起動」ダイアログが出たら「OK」を押すとソースコードが保存されて実行される。


(12) 画面下の「コンソール」に「Hello」と表示されるのを確認する。
