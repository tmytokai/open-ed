クラスの追加

プロジェクトに「クラス」を新規に作成追加してそれを利用する方法を学びます。

(1) クラスの新規作成

「クラス」が一体何かはおいおい説明しますので、とりあえず下の手順で新しいクラスを作ってみてください。


(1) 画面左側にある「パッケージ・エクスプローラ」の「デフォルトパッケージ」を右クリックして「新規」→「クラス」を選ぶ。


(2) 「新規Javaクラス」ダイアログが出たら「名前」に「Test」と入れる。もし「どのメソッド・スタブを作成しますか？」の「public static void main(String[] args)」にチェックが入っていたら外す。最後に「完了」を押す。


(3)もしソースコード(Test.java)が表示されなかったり、間違ってタブを閉じてしまった場合は、パッケージ・エクスプローラの三角をクリックして行ってTest.javaを見つけてダブルクリックする。


(2) クラスの利用

以上で「Test」という新しいクラスができますが、このままだと何も起きませんので、Testクラスに何かメソッド(関数)を追加して main メソッドからそのメソッドを利用してみましょう。


(1) 「public class Test」の中括弧の間に以下の 「test」 メソッド(関数)を追加する。 
public void test()
{
	System.out.println("Testクラスです");
}

(2) main メソッドを開き、以下のコードを追加する。 
Test t = new Test();
t.test();

(3) 実行すると main メソッドから Test クラスの test メソッドが呼び出されていることが確認できる。
