4. 例1: 有限長の時間領域ディジタル信号


演習 1 (個人ノート、15分): $f[i]$ が有限長である場合の Z 変換を求めてみましょう。

$\textrm{L} = 2$、$f[i] = \{1, 2, 0, 0, 0, \cdots \}$ の Z 変換 $\textrm{F}(z)$ の式をノート上で求めてください。
Z 平面上の収束領域をノートに図示して下さい。
$z=2j$ の位置を収束領域の図に追加し、この点が収束領域に含まれるか含まれないかをノートに書いて下さい。
$\textrm{F}(2j)$ の値をノート上で求めてください。
ノートに複素平面と $\textrm{F}(2j)$ の位置を描き、その絶対値および偏角を求めて下さい。
