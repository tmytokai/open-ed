\noindent \textbf{【出題意図】}

\bigskip
\noindent ディジタルインパルス信号 $\delta[i] = \{1,0,0,\cdots \}$ の Z 変換の具体的な値を求めることができるかどうかを確かめる問題である。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【重要事項】}

\medskip
$z$ の値によらず $\textrm{F}(z) = 1$ となる。

\bigskip

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【解説】}

\bigskip
\noindent 重要事項より、常に $\textrm{F}(z) = 1$ となる。
