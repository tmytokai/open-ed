1. 周期性時間領域ディジタル信号


演習 1-1 (個人): 表計算ソフトを使って周期性時間領域ディジタル信号

\[ f[i] = -1 + 2 \cdot \cos \left ( \frac{2 \pi}{ \left ( \frac{15}{1} \right ) } \cdot i + 1 \right ) + 4 \cdot \cos \left ( \frac{2 \pi}{ \left ( \frac{15}{3} \right ) } \cdot i +0.5 \right ) + 3 \cdot \cos \left ( \frac{2 \pi}{ \left ( \frac{15}{7} \right ) } \cdot i -2 \right ) \]

のグラフを描いてみましょう。

表計算ソフトを起動します。
時刻の範囲は $i = 0, 1, \cdots, 44$ とします。
一番上の最初の列のセルに「i」と書き、その下のセルから各 $i$ の値をオートフィルを使って入力します。
「i」と書いた右隣のセルに「f[i]」と書き、その下のセルから各 $i$ における $f[i]$ の値をオートフィルを使って入力します。
「i」と「f[i]」の列を範囲選択してからグラフを挿入します。グラフの種類は散布図、点あり、線あり、平滑化なしとします。
グラフを見て $f[i]$ の周期 $\textrm{N}$ [点]を求めて下さい。
シートを保存します。ファイル名は「DFT_1_1」として下さい(拡張子は表計算ソフトによる)。
チーム内で答え合わせをして下さい。
指定場所に「周期 N [点]」と「グラフ」を貼り付けて下さい。

3. プログラミング


演習 1-3 (個人): 演習 1-1 の時間領域ディジタル信号

\[ f[i] = -1 + 2 \cdot \cos \left ( \frac{2 \pi}{ \left ( \frac{15}{1} \right ) } \cdot i + 1 \right ) + 4 \cdot \cos \left ( \frac{2 \pi}{ \left ( \frac{15}{3} \right ) } \cdot i +0.5 \right ) + 3 \cdot \cos \left ( \frac{2 \pi}{ \left ( \frac{15}{7} \right ) } \cdot i -2 \right ) \]

の DFT 係数 $\textrm{C}[k]$ の絶対値 $|\textrm{C}[k]|$ と偏角 $\angle \textrm{C}[k]$ をプログラムで求め、表計算ソフトでグラフ化しましょう。

フォルダ「DFT_1_3」を新規作成し、その中に移動します
フォルダ「DFT_1_3」の中に Main.cpp を作成します
Main.cpp を開き、ファイルの中身をテンプレートの内容に置き換えて下さい。
テンプレートソースを開いて穴埋めをします。
コンパイルして実行するとCSVファイル(DFT_1_3.csv)が出力されます。1列目がDTF係数の番号 $k$、2列目が絶対値 $|\textrm{C}[k]|$、3列目が偏角 $\angle \textrm{C}[k]$ です。
CSV ファイルを表計算ソフトで開き、一行目に空行を挿入し、最初の列の先頭に「k」、次の列の先頭に「|C[k]|」、次の列の先頭に「∠C[k]」と入力して下さい。
1列目と2列目を選択して絶対値 $|\textrm{C}[k]|$ のグラフを挿入します。グラフの種類は散布図、点あり、線なし、平滑化なしとします。
1列目と3列目を選択して偏角 $\angle \textrm{C}[k]$ のグラフを挿入します。グラフの種類は散布図、点あり、線なし、平滑化なしとします。
「名前を付けて保存」でシートを表計算形式で保存します。ファイル名は「DFT_1_3」として下さい(拡張子は使用する表計算ソフトによる)。
チーム内で答え合わせをして下さい。
指定場所に「ソースコード」「絶対値のグラフ」「偏角のグラフ」を貼り付けて下さい。
