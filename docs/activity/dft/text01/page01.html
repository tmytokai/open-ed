まず DFT の大前提となる周期性時間領域ディジタル信号について説明します。

周期性時間領域ディジタル信号とはある周期ごとに同じ波形を繰り返すディジタル信号のことで、定義は以下のようになります。

定義： 周期性時間領域ディジタル信号 $f[i]$

ある正整数 $\textrm{N}$ が与えられた時

\[ f[i] = f[i + n \cdot \textrm{N}] \]

の関係を満たすディジタル信号を周期性ディジタル信号という。

$\textrm{N}$ ・・・ 周期、正整数の定数、単位は[点]

$n$ ・・・ 任意の整数 $(n = 0, \pm 1, \pm 2, \cdots)$

$i$・・・ディジタル時刻、整数の変数、単位は無し(※ ディジタル信号なので秒の概念は無い)

なお周期 $\textrm{T}_d$ のディジタルサイン波は周期 $\textrm{N} = \textrm{T}_d$ の周期性時間領域ディジタル信号でもあります。
