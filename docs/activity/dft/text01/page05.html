1. アクティビティ: 離散フーリエ変換 (DFT)
  2. 学習項目: [1] DFT と IDFT

DFT / IDFT の最大の欠点は計算量が多くて時間がかかることです。具体的な DFT / IDFT の計算量(オーダー)は次の通りです。

DFT / IDFT の計算量(オーダー)

$\textrm{N}$ を周期とした時

\[ \textrm{O}(\textrm{N}^2) \]

そこで DFT を高速化した高速フーリエ変換( Fast Fourier Transform, FFT )、及び高速フーリエ逆変換( Inverse Fast Fourier Transform, IFFT )が考案されました。

FFT / IFFT は周期 $\textrm{N}$ が $2$ の累乗(例えば $\textrm{N} = 2, 4, 8, 16, 32, 64, \cdots$ )でないと使えないという制限はありますが、次の様に計算量が劇的に減ります。もちろん計算結果は DFT / IDFT と同一です。

FFT / IFFT の計算量(オーダー)

$\textrm{N}$ を周期とした時(※)

\[ \textrm{O}(\textrm{N}\log_2\textrm{N}) \]

※ ただし $\textrm{N}$ は $2$ の累乗(例えば $\textrm{N} = 2, 4, 8, 16, 32, 64, \cdots$ ) である必要がある。

$\textrm{N}$ が小さい時は DFT と FFT はそれ程計算速度は変わりませんが、大きくなるにつれて速度差が大きくなっていきます(演習で確認します)。
従って世間一般的には DFT は普通使わず、代わりに FFT が良く使われています。
