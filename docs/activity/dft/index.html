目的

あなたが周期性時間領域ディジタル信号に対して離散フーリエ変換(DFT)を行える技術者になること

必要性

離散フーリエ変換 (DFT) はディジタル信号版の複素フーリエ級数展開です。
従ってディジタル信号のスペクトル解析を学ぶために、技術者は離散フーリエ変換 (DFT) を学ぶ必要があります。

進め方

上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
アクティビティの具体的な進め方についてはこちらを参照して下さい。
