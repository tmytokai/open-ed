左右に動かす

今のままでは Ball は全く動きませんので、Ball に 「Rigidbody2D」コンポーネントをアタッチしてから「velocity」プロパティに方向と速さを指定してあげます。

velocity を指定する場所ですが、今回は一度設定してしまえば途中で値を変更する必要が無いので Ball の Start イベントの中でセットしてあげれば良いと思います。

もちろん、ホーミング弾みたいに実行中に方向や速さが変わる場合は Update イベントの中で velocity の値を随時変更します。
