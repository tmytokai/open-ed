Blast の表示

最後の仕上げとして、Bat が Cat や Ball に当たった時に爆風(Blast)を表示してみましょう。

と言っても、やることは

(1) Ball が Bat に当たったら Ball の「OnTriggerEnter」イベントの中で Blast をインスタンス化する

(2) Cat が Bat に当たったら Cat の「OnTriggerEnter」イベントの中で Blast をインスタンス化する

(3) Blast の Start イベントの中で Blast を自動で消す

だけです。
