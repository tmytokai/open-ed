当たり判定

このページでは Ball と Bat 間、及び Cat と Bat 間に当たり判定を付けます。

ゲームオブジェクト間の当たり判定を行うためには Collider コンポーネントをアタッチするのですが、実際の Unity では当たり判定の方法はかなり複雑です。

基本的に当たり判定の仕方は「OnCollision 系列」と呼ばれる方法と「OnTrigger 系列」と呼ばれる方法の２通りに分かれているのですが、どちらの方法を使えば良いのかは、そのオブジェクトに Rigidbody2D がアタッチされているか、トリガーかどうか(Collider の isTrigger プロパティで切り替え)、物理エンジンを使用しているかどうか(Rigidbody2D の isKinematic プロパティで切り替え)等の条件で変化します。

従って今回の scratch2unity ではこの辺りを思いっきり簡略化し、

(1) Ball が Bat に当たったら Ball の「OnTriggerEnter」イベントが呼び出される

(2) Cat が Bat に当たったら Cat の「OnTriggerEnter」イベントが呼び出される

ことにしています。

当たり判定方法に関して詳しくは、本格的に Unity を使う際に自分で調べてみてください。
