プロパティとメソッド

scratch2unity の変数がそのまま Unity の「プロパティ」に相当します。

ただし今回の演習では(コンポーネントのプロパティ以外の)プロパティを使いませんので、これ以上のプロパティについての説明は省略します。
コンポーネントのプロパティの取り扱いについては演習を進めながらその都度説明します。

一方、C# と Scratch の違いで一番問題となるのが Unity の「メソッド」の取り扱い方です。

実は Scratch 1.4 には関数の概念が無いので(※)そのままではメソッド、特に「イベントメソッド」を扱うことが出来ません。

※ Scratch 2系にはあります。

ただしプロパティと同様に、(イベントメソッド以外の)メソッドも今回の演習では使いませんので、これ以上のメソッドについての説明は省略します。
イベントメソッドの取り扱いについては「イベント」のページで改めて説明します。
