まとめ

プロパティとメソッド

・各ゲームオブジェクトに「プロパティ」と「メソッド」を持たせることが出来る

・「プロパティ」はそのゲームオブジェクトの属性や性質を表す情報のこと

・「メソッド」はゲームオブジェクトに対して様々な操作をするためのプログラムをひとつにまとめた塊(ルーチン)のこと

・プロパティの値のセット/取得方法

セット: ゲームオブジェクト名.プロパティ = セットしたい値 ;
取得: 変数名 = ゲームオブジェクト名.プロパティ ;
※ .(ドット) を「ドット演算子」と言う

・メソッドの呼び出し方法

変数名 = ゲームオブジェクト名.メソッド名( 引数1, 引数2, … ) ;

・用語の読み替え方

プロパティ ＝ (メンバ)変数 ＝ フィールド(※)
メソッド ＝ (メンバ)関数

(※) フィールドは文字通り「ゲームの背景・領域・土地・場所」の意味で使っている文献も多い


コンポーネント

・「コンポーネント(Component)」はゲームオブジェクトに性質や動き方を設定するための部品のこと

・プログラマーが簡単にゲームを作れるように、良く使うプロパティやメソッドをグループ化してまとめている

・特性により複数の種類にグループ化されて分けられている

・ゲームオブジェクトに何かのコンポーネントを持たせることを「アタッチする」という
