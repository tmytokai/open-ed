アクティビティ: Scratch による Unity プログラミング入門

目的

あなたが Scratch を使って Unity の流儀に基づいたゲームプログラミングを学ぶこと


必要性

Unity は世界的に使われている高機能なゲームエンジンですが、スクリプト用言語として一般に C# が用いられているためプログラミング初心者にとっては敷居が高いです。一方 Scratch はプログラミング初心者にとってとっつきやすいプログラミング学習環境です。
従って Unity を使って本格的にゲームプログラミングをする前に、Scratch を使ってUnity の流儀に基づいたゲームプログラミングに技術者は慣れる必要があります。
