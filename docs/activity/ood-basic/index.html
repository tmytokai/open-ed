最終更新日: 2017/8/28

目的

あなたがミニゲーム作りを通じて初歩的なオブジェクト指向開発について学ぶこと

必要性

オブジェクト指向型プログラミング言語を用いたソフトウェア開発では反復型開発手法が用いられる事が多いのですが、開発未経験者がいきなり本格的なオブジェクト指向開発を行うのは大変です。
従って本格的なオブジェクト指向開発を行う前に、初歩的なオブジェクト指向開発に技術者は慣れる必要があります。

進め方

もし上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
またこのアクティビティの具体的な進め方についてはこちらを参照して下さい。
