目的

あなたがGreenfootを用いたミニゲーム作りを通じてオブジェクト指向開発の基礎について学ぶこと

必要性

オブジェクト指向型プログラミング言語によるソフトウェア開発ではスクラムなどのアジャイル開発手法が用いられる事が多いです。
従ってアジャイル開発手法を用いたソフトウェア開発について技術者は学ぶ必要があります。

進め方

上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
アクティビティの具体的な進め方についてはこちらを参照して下さい。
