1. スプリント プランニング

詳細設計:

このスプリントでは、「HANAKO」が「APPLE」と「ORANGE」を全て食べたらメッセージを表示してゲームクリアにします。

  • メッセージの内容は任意とします。

スプリント バックログ:

  1. 当たり判定とゲームクリア処理

2. 開発作業

準備:

プロジェクト「UNITY-2」に対して作業を続けます。


スプリントバックログ 1 : 当たり判定とゲームクリア処理

「HANAKO」スクリプトをエディタで開き、 using 部に「using UnityEngine.UI;」を追加します。
食べた果物の数をカウントする「private int fruits;」 フィールドを追加し、Start メソッドで 0 クリアして下さい。
void FixeUpdate() の下に void OnCollisionEnter2D( Collision2D collision ) メソッドを追加して下さい。
OnCollisionEnter2D メソッド内に、「TagFruit というタグを持つオブジェクトと衝突したらそのオブジェクトを消して fruits の数をひとつ増やし、fruits が 10 になったら MessageText にメッセージを表示してプログラムを停止する」プログラムを記述して下さい。


テストとデバッグ:

実行ボタンを押して動作確認して下さい。
上手く動作しなかった場合は実行ボタンを押して動作を止めてデバッグして下さい。


3. リリース

チーム内で答え合わせをし、提出用のスクリプト(HANAKO.cs)を選んで下さい。
指定場所に「ソースコード」を貼り付けて下さい。
