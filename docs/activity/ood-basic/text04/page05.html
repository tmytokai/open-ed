1. スプリント プランニング

詳細設計:

このスプリントでは、「APPLE」と「ORANGE」を作成し、Prefab化して MyWorld に初期配置します。

  • APPLE、ORANGE の画像は任意とします。

また Prefab のインスタンス化は MyWorld のスクリプトの中で行います。

  • Start メソッドの中でそれぞれ 5 個ずつインタンス化します。
  • 位置と回転方向はランダムです。

スプリント バックログ:

  1. 「APPLE」「ORANGE」オブジェクトを作成して Prefab 化
  2. 「APPLE」「ORANGE」のインスタンス化

2. 開発作業

準備:

プロジェクト「UNITY-2」に対して作業を続けます。


スプリントバックログ 1: 「APPLE」「ORANGE」オブジェクトを作成して Prefab 化

画像を用意し「APPLE」に名前変更して下さい(※ 拡張子は変更しないよう注意)。
画像を「APPLE」オブジェクト化して下さい。サイズは適宜変更して下さい。位置は適当で結構です。
「APPLE」オブジェクトに Rigidbody 2D をアタッチして重力(Gravity Scale)を 0 にして下さい。Constraints → Freeze Rotation → Z にチェックは入れないで下さい。
「APPLE」オブジェクトに Circle Collider 2D をアタッチして下さい。衝突範囲は適宜調整して下さい。
「APPLE」を Prefab 化し、オブジェクトを削除して下さい。
同様に「ORANGE」の Prefab も作成して下さい。


スプリントバックログ 2 : 「APPLE」「ORANGE」のインスタンス化

「MyWorld」スクリプトを新規作成して MyWorld にアタッチして下さい。
「MyWorld」スクリプトを開き、以下のフイールドを追加して下さい。

public GameObject apple;
public GameObject orange;

Start メソッドの中で「APPLE」と「ORANGE」をそれぞれ 5 つ、計 10 個インスタンス化して下さい。
表示位置は以下のようにランダムとします。

float x = Random.Range( 最小の横座標, 最大の横座標 ); // Random.Range は指定した範囲の乱数を返す
float y = Random.Range( 最小の縦座標, 最大の縦座標 );
Vector2 position = new Vector2 ( x, y );

なお乱数の範囲は本来ならカメラから正確な表示範囲の座標を取得するべきですが、今回は適当に決めて結構です(Sceneビューで TARO を動かして Inspector の Position の x, y を見ると楽に範囲が分かります)。
また回転角度は

Quaternion rotation = Quaternion.Euler( 0f, 0f, Random.Range( 0, 360 ) );

で0度から360度までランダムにセットできます。

スクリプトを保存して Unity エディタに戻って下さい。
MyWorld の Inspector ウィンドウを開き、MyWorld(Script) の Apple と Orange に「APPLE」「ORANGE」Prefab　を DnD して下さい。


テストとデバッグ:

実行ボタンを押して動作確認して下さい。
上手く動作しなかった場合は実行ボタンを押して動作を止めてデバッグして下さい。


3. リリース

チーム内で答え合わせをし、提出用のスクリプト(MyWorld.cs)を選んで下さい。
指定場所に「ソースコード」を貼り付けて下さい。
