プログラム内で指定したオブジェクトを削除には次の Destroy メソッドを使います。

オブジェクトを削除 
public static void Destroy( オブジェクト, 削除するまでの時間);

戻り値(void): 無し

オブジェクト: 削除したいオブジェクト
削除するまでの時間(秒): 指定した秒数後に削除する。指定しなくても良い(その場合は即削除)

問題はどうやって削除したいオブジェクトを取得するかですが、自分自身を削除したい場合は次のようにして簡単に出来ます。

自分自身を削除 
Destroy( gameObject );

また、衝突時にぶつかったオブジェクトを削除したい場合も簡単に出来ます。

OnCollisionEnter2D の場合は

OnCollisionEnter2D で衝突したオブジェクトを削除 
void OnCollisionEnter2D( Collision2D collision )
{
    if (collision.gameObject.tag == "TagA") {	

       // TagA がついたオブジェクトと衝突したらそのオブジェクトを削除
       Destroy (collision.gameObject);
    }

}

OnTriggerEnter2D の場合は

OnTriggerEnter2D 
void OnTriggerEnter2D( Collider2D collider ) // OnCollisionEnter2D と引数の型が異なることに注意
{
    if ( collider.tag == "TagA") {	

       // TagA がついたオブジェクトと重なったらそのオブジェクトを削除
       Destroy (collider.gameObject);
    }
}
