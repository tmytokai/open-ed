次にキャラクターのオブジェクトを動かしてみたいと思います。

オブジェクトを動かす方法は色々ありますが、今回は物理演算エンジンを使って動かしてみたいと思います。
そのためにはまず、以下の手順で「Rigidbody 2D」というコンポーネントをオブジェクトにアタッチします(※)。

※ コンポーネント(Component)は各オブジェクトの振る舞いを定めるメソッドやプロパティのグループのことです。
コンポーネントをオブジェクトにくっつけることをアタッチと言います。
例えばこれまでのページで使ってきた Transform も、アタッチされているオブジェクトの位置やサイズを定めるコンポーネントのひとつです。
Rigidbody 2D はアタッチするオブジェクトの速度や重さ、重力などを定めるコンポーネントです。

Rigidbody 2D コンポーネントのアタッチ方法:

対象のオブジェクトを選択する。
Inspector ウィンドウの Add Component ボタンを押してメニューを表示する。
Physics 2D → Rigidbody 2D を選択すると Inspector ウィンドウに Rigidbody 2D の項目が現れる。
重力が必要ない場合は Rigidbody 2D の Gravity Scale の値を 0 にする。
他のオブジェクトと衝突した時に回転させたくない場合は、Constraints → Freeze Rotation → Z をチェックする。

ただし Rigidbody 2D をアタッチしただけでは任意の方向にオブジェクトを動かす事は出来ませんので、次は「スクリプト」コンポーネントもアタッチします(※)。

※ スクリプトの実体は C# 又は JavaScript で書かれたスクリプトファイルです。今回は C# の方を使います。

スクリプト コンポーネントのアタッチ方法:

Assets フォルダで右クリックし、Create → C# Script を選ぶとスクリプトファイルが出来るので、適切なファイル名に変更する。
スクリプトをアタッチしたいオブジェクトを選択する。
Assets フォルダの中にある先ほど作成したスクリプトファイルを Inspector ウィンドウの上にドラッグしてドロップするとアタッチされる。
Assets フォルダの中のスクリプトファイルをダブルクリックするとエディタが起動する。
スクリプト内の Start メソッドはオブジェクトが表示される前に一度だけ呼び出される初期化用メソッドで、Update は描画フレームごとに繰り返し呼び出されるメソッドである。

では準備が済んだので実際にオブジェクトを動かしてみましょう。

オブジェクトを動かすためには Rigidbody 2D コンポーネント の velocity (ベロシティ) プロパティに値をセットします。
この velocity は日本語では「速度」を意味する 2 次元ベクトルで、更に「velocity.x」と「velocity.y」というプロパティに分かれます。
velocity.x は横方向の速度、velocity.y は縦方向の速度であり、それらに指定した方向と速さでオブジェクトが動きます(図1)。

図1. Rigidbody 2D の velociy で指定した方角にオブジェクトが動く

後は以下の様にしてスクリプトの中で velocity の値を変更すればオブジェクトは指定したスピードと方角で移動します。

オブジェクト移動 
float speed = スピード;
float x = 横方向の移動量;
float y = 縦方向の移動量;
Vector2 v = new Vector2( x, y ).normalized; // ※ .normalized はベクトル v を長さ 1 に正規化するの意味
GetComponent<Rigidbody2D>().velocity = v * speed;

なお、摩擦が無い限り一度 velocity を設定すればその方向に動き続けます。
また、重力が 0 で無い時は重力の影響も受けますので、想定した方向には動きませんのでご注意下さい。

例えばあるオブジェクトを(重力が 0 なら) 斜め右上 45 度の方向に速度 2.0 で動かしたい場合はソース 1 の様にスクリプト内に書きます。

ソース 1: オブジェクトを斜め右上 45 度の方向に速度 2.0 で動かす 
float speed = 2.0f;
float x = 1.0f;
float y = 1.0f;
Vector2 v = new Vector2( x, y ).normalized; // ※ .normalized はベクトル v を長さ 1 に正規化するの意味
GetComponent<Rigidbody2D>().velocity = v * speed;

さてコーディングが済んだら以下の手順でプログラムを実行してオブジェクトを動かしてみましょう。

プログラム実行方法:

スクリプトを上書き保存すると自動でコンパイルされる。
Unity エディタの画面に戻り、上部にある再生ボタンを押すと GAME ビューに画面が切り替わってプログラムが実行される。
もう一度再生ボタンを押すと停止する。
