1. Unity2D とは

演習無し


2. ブロジェクトの新規作成とシーン作成

演習 1-1 (個人PC、10 分):

Unity を起動します。
新規プロジェクトを作成します。プロジェクト名は「UNITY-1」として下さい。また開発モードは「2D」にして下さい。
適当な背景画像をネットから探し、「MyWorld」に名前変更して下さい(※ 拡張子は変更しないよう注意)。
背景を Scene ビューに表示してからカメラの調節をして下さい。画面比率は 5:4 とします。
続けて次の演習を行うので提出物はありません。

※ 途中でデフォルトシーンのファイル名が尋ねられた場合、ファイル名は「MyWorld」として下さい。


3. オブジェクトの作成と配置

演習 1-2 (個人PC、10 分):

プロジェクト「UNITY-1」に対して作業を続けます。
適当な画像をネットから探し、「TARO」に名前変更して下さい(※ 拡張子は変更しないよう注意)。
画像をオブジェクト化し、サイズと位置を調整して下さい。
「TARO」オブジェクトをコピーして「TARO2」オブジェクトを作って下さい。
「TARO2」オブジェクトを削除して下さい。
続けて次の演習を行うので提出物はありません。


4. オブジェクトの移動

演習 1-3 (個人PC、10 分):

プロジェクト「UNITY-1」に対して作業を続けます。
TARO オブジェクトに Rigidbody 2D コンポーネントをアタッチし、重力を 0 にしてください。
TAROという C# のスクリプトファイルを作成して TARO オブジェクトにアタッチして下さい。
TAROスクリプトを開き、Start メソッド内に、「TARO が斜め右下 45 度の方向に速度 0.5 で動く」プログラムを記述して下さい。
スクリプトを保存してコンパイルして下さい。
再生ボタンを押して動作確認して下さい。
続けて次の演習を行うので提出物はありません。
