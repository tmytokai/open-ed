2. ブロジェクトの新規作成とシーン作成

7. 衝突判定処理

演習 1-7 (個人PC、10 分):　 OnCollisionEnter2D の使い方を演習します。

プロジェクト「UNITY-1」に対して作業を続けます。
「TARO」オブジェクトに Box Collider 2D コンポーネントをアタッチし、衝突領域を適宜調整して下さい。
適当な画像をネットから探し、「HANAKO」に名前変更して下さい(※ 拡張子は変更しないよう注意)。
画像から「HANAKO」オブジェクトを作成し、サイズと位置を適宜調整して下さい。
「HANAKO」オブジェクトに Box Collider 2D コンポーネントをアタッチし、衝突領域を適宜調整して下さい。
この時点で一旦再生ボタンを押し、TARO と HANAKO オブジェクトが衝突して重ならない事を確認して下さい。
プログラムを停止し、TARO スクリプトをエディタで開き、FixedUpdate メソッドの下に void OnCollisionEnter2D( Collision2D collision ) メソッドを追加して下さい。
OnCollisionEnter2D メソッド内に、「他のオブジェクトと衝突したら Text1 に "ITAI" と表示する」プログラムを記述して下さい。
スクリプトを保存してコンパイルして下さい。
再生ボタンを押して動作確認して下さい。
続けて次の演習を行うので提出物はありません。

8. prefab とオブジェクトの動的配置

演習 1-11 (個人PC、10 分):

My World(Script) コンポーネントの Taro のエリアに Assets フォルダから TARO Prefab を DnD します。
再生ボタンを押して動作確認して下さい。
