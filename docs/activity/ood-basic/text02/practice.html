1. Unity2D とは

2. ブロジェクトの新規作成とシーン作成

演習 1-1 (個人PC、10 分):

Unity を起動します。
新規プロジェクトを作成します。プロジェクト名は「UNITY-1」として下さい。また開発モードは「2D」にして下さい。
適当な背景画像をネットから探し、「MyWorld」に名前変更して下さい(※ 拡張子は変更しないよう注意)。
背景を Scene ビューに表示してからカメラの調節をして下さい。画面比率は 5:4 とします。
続けて次の演習を行うので提出物はありません。

※ 途中でデフォルトシーンのファイル名を尋ねられた場合、ファイル名は「MyWorld」として下さい。


3. オブジェクトの作成と配置

演習 1-2 (個人PC、10 分):

プロジェクト「UNITY-1」に対して作業を続けます。
適当な画像をネットから探し、「TARO」に名前変更して下さい(※ 拡張子は変更しないよう注意)。
画像から「TARO」オブジェクトを作成し、サイズと位置を適宜調整して下さい。
「TARO」オブジェクトをコピーして「TARO2」オブジェクトを作って下さい。
「TARO2」オブジェクトを削除して下さい。
続けて次の演習を行うので提出物はありません。


4. オブジェクトの移動

演習 1-3 (個人PC、10 分):

プロジェクト「UNITY-1」に対して作業を続けます。
TARO オブジェクトに Rigidbody 2D コンポーネントをアタッチし、重力を 0 にし、Freeze Rotation の z をチェックしてください。
「TARO」という C# のスクリプトファイルを新規作成して TARO オブジェクトにアタッチして下さい。
TARO スクリプトをエディタで開き、Start メソッド内に「TARO が斜め右下 45 度の方向に速度 0.5 で動く」プログラムを記述して下さい。
スクリプトを保存してコンパイルして下さい。
再生ボタンを押して動作確認して下さい。
続けて次の演習を行うので提出物はありません。


5. ゲームパッドやキーボードによるオブジェクトの移動

演習 1-4 (個人PC、10 分):

プロジェクト「UNITY-1」に対して作業を続けます。

FixedUpdate メソッド内に、「方向ボタンで入力した方向に速度 0.5 でTARO が動く」プログラムを記述して下さい。

再生ボタンを押して動作確認して下さい。
続けて次の演習を行うので提出物はありません。

FixedUpdate メソッド内に、「Jump ボタン(デフォルトではスペースキー)を押している間、TARO が斜め右下 45 度の方向に速度 0.5 で動く」プログラムを記述して下さい。

6. 文字列の表示

演習 1-6 (個人PC、10 分):

プロジェクト「UNITY-1」に対して作業を続けます。
「MyCanvas」という Canvas 型オブジェクトと「Text1」「Text2」という２つの Text 型オブジェクトを新規作成します。
「Text1」「Text2」の位置やフォントサイズ等を適宜調整します。
TARO スクリプトをエディタで開き、using 部に「using UnityEngine.UI;」を追加します。
演習 1-5 で書いた部分をコメントアウトし、代わりに演習 1-4 で書いた部分(上下移動)のコメントアウトを解除します。
TARO スクリプトの FixedUpdate メソッド内に、「Jump ボタン(デフォルトではスペースキー)を押したら Text1 に "HOGE"、Fire1 ホダン(デフォルトでは左 Ctrl キー)を押したら Text2 に "HELLO" と表示する」プログラムを記述して下さい。
スクリプトを保存してコンパイルして下さい。
再生ボタンを押して動作確認して下さい。
続けて次の演習を行うので提出物はありません。


7. prefab とオブジェクトの動的配置

演習 1-7 (個人PC、10 分):

プロジェクト「UNITY-1」に対して作業を続けます。
「TARO」オブジェクトの Prefab を作り、Hierarchy ウィンドウから TARO オブジェクトを削除して下さい。
「MyWorld」という C# のスクリプトファイルを新規作成して MyWorld オブジェクトにアタッチして下さい。
MyWorld スクリプトをエディタで開き、「public GameObject taro;」フィールドを追加します。
Start メソッドの中に「TARO を座標 (0.1f, -0.1f) および (-0.1f, 0.1f) に無回転で 2 体表示する」プログラムを記述して下さい。
スクリプトを保存して Unity エディタに戻ったらHierarchy ウィンドウの MyWorld オブジェクトをクリックして Inspector ウィンドウを表示します。
My World(Script) コンポーネントの Taro のエリアに Assets フォルダから TARO Prefab を DnD します。
再生ボタンを押して動作確認して下さい。
続けて次の演習を行うので提出物はありません。
