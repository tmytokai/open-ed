MyWorld の設定が済んだので、次はキャラクターなどのオブジェクトを作成して MyWorld の上に配置してみましょう。

Greenfoot は Java がベースとなっていますので、オブジェクトを作成するにはまず元となるクラスを作る必要があります。
さらにそのクラスはクラス画面上に既に表示されている Actor クラスのサブクラスである必要があります。
Actor クラスのサブクラスを作るには、クラス画面にある「Actor」クラスの上で右クリックしてメニューを表示し、New Subclass を選びます。
するとダイアログが表示されますので、クラス名と画像を選択して下さい。

こうやって新規作成したキャラクターのクラスから手動でオブジェクトを作ってワールドに配置する手順は以下のとおりです。

※ new を使って動的に配置する方法は7ページ目で説明します

オブジェクトを手動で配置する方法:

表示したいキャラクターのクラスをクラス画面からクリックで選択する
シフトキーを押すとカーソルの位置にオブジェクトが表示される
配置したい場所にカーソルを移動し、「シフトキーを押しながら」クリックする

なお手動配置したオブジェクトを削除したい場合は、削除したいオブジェクトの上で右クリックしてメニューを表示し、Delete を選んで下さい。
