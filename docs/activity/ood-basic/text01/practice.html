1. Greenfoot とは

2. シナリオの新規作成とワールド作成

演習 1-1 (個人PC、10 分):

Greenfoot を起動します。
新規 Java シナリオを作成します。シナリオ名は「GF-1」として下さい。
(MyWorldが無ければ) 「World」クラスを継承して「MyWorld」クラスを作成して下さい。
適当な背景をセットし、大きさを 300x200 に変更して下さい。
続けて次の演習を行うので提出物はありません。


3. オブジェクトの作成と配置

演習 1-2 (個人PC、10 分):

シナリオ「GF-1」に対して作業を続けます。
「Actor」クラスを継承して「TARO」クラスを作成して下さい。
「TARO」オブジェクトをひとつだけワールドにマウスで配置して下さい。
続けて次の演習を行うので提出物はありません。


4. オブジェクトの移動

演習 1-3 (個人PC、10 分):

シナリオ「GF-1」に対して作業を続けます。
「TARO」サブクラスの act メソッド内に、「TARO が下に 50 歩動く」プログラムを記述して下さい。
Act ボタンを１回押して動作確認して下さい。
同様に Run ボタンを１回押して動作確認して下さい。
続けて次の演習を行うので提出物はありません。


演習 1-4 (個人PC、10 分):

シナリオ「GF-1」に対して作業を続けます。
演習 1-3 で書いた部分をコメントアウトします。
次は 「Act ボタンを押すたびにTAROの方向が 0 度、90度、270度、0度・・・と回転し、回転した方向に 50 歩動く」プログラムを記述して下さい。
Act ボタンを１回押して動作確認して下さい。
同様に Run ボタンを１回押して動作確認して下さい。
続けて次の演習を行うので提出物はありません。


5. キーボードによるオブジェクトの移動

演習 1-5 (個人PC、10 分):

シナリオ「GF-1」に対して作業を続けます。
演習 1-4 で書いた部分をコメントアウトします。
次は 「キーボードの右矢印キーを押すと TARO が右に 50 歩、左矢印キーを押すと TARO が左に 50 歩動く」プログラムを記述して下さい。
Run ボタンを１回押して動作確認して下さい。
続けて次の演習を行うので提出物はありません。


6. 文字列の表示

演習 1-6 (個人PC、10 分):

シナリオ「GF-1」に対して作業を続けます。
演習 1-6 で書いた部分の続きにコードを追加していきます。
次は 「キーボードの a キーを押すと 100, 100 座標に文字列「Hello」と表示し、s キーを押したら同じ座標に "konnitiwa"と表示し、"d"キーを押すと文字列を消す」プログラムを記述して下さい。
Run ボタンを１回押して動作確認して下さい。
続けて次の演習を行うので提出物はありません。
