プログラムを動作を停止するには、以前も出てきた Greenfoot クラスの stop メソッドを利用します。

プログラムの停止 
public static void Greenfoot.stop()

戻り値(void): 無し。

※ プログラムを再開させるには Reset ボタンを押してからもう１回 Run ボタンを押す。

このメソッドは public static なので、ソース 1 のようにしてコード内のどこからでも呼び出すことが出来ます。

ソース 1: プログラムを停止する例 
Greenfoot.stop();
