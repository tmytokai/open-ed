ワールド上に文字列を表示したい場合は、World クラスの showText メソッドを利用します。

文字列表示 
public void World.showText( "文字列", x 座標, y 座標 ) 

戻り値(void): 無し。

同じ座標に違う文字列を上書き表示したいなら、座標はそのままで文字列だけ変える
文字列を消すには同じ座標で "" を表示する

ただし、あるクラスのメソッドの中で文字列を表示させたい場合は、そのオブジェクトが表示されているワールドのオブジェクトを取得しなければいけません。
それにはワールドオブジェクトの getter である getWorld メソッドを利用します。

例えばあるクラスの中で文字列 "HOGE" を座標 (100,50) に表示するには ソース 1 の様にします。

ソース 1: 文字列 "HOGE" を座標 (100,100) に表示 
getWorld().showText( "HOGE", 100, 50 ); // getWorld() は World オブジェクトの getter

ちなみに MyWorld の様な World クラスのサブクラスの場合は showText が継承されていますので、ソース 2 の様に getWorld を使わず直接 showText を呼び出せます。

ソース 2: World のサブクラス内で文字列を表示 
showText( "HOGE", 100, 50 );
