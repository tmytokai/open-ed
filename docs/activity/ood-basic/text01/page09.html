オブジェクトをプログラム内で動的に削除するには、World クラスで定義されている removeObject メソッドを利用します。

Actor のサブクラスのオブジェクトを削除 
public void World.removeObject( オブジェクト )

戻り値(void): 無し。

このメソッドを使うためには6ページで説明した getWorld メソッドを使ってワールドオブジェクトを取得する必要があります。
さらに削除対称のオブジェクトも取得する必要があります。

削除対象のオブジェクトを取得する方法は色々ありますが、例えばあるオブジェクトが TARO オブジェクトにぶつかったら TARO オブジェクトを削除する処理はソース 1 の様になります。

ソース 1: あるオブジェクトが TARO オブジェクトにぶつかったら TARO オブジェクトを削除 
Actor actor = getOneIntersectingObject( TARO.class );
if( actor != null ){
    getWorld().removeObject( actor );
}
