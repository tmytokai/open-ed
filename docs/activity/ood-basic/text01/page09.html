Actor のサブクラスのオブジェクトをプログラム内で動的に削除するには、World クラスの removeObject メソッドを利用します。

Actor のサブクラスのオブジェクトを削除 
public void World.removeObject( オブジェクト )

戻り値(void): 無し。

ただし、このメソッドを使うには削除したいオブジェクトを取得する必要があります。

削除対象のオブジェクトを取得する方法は色々ありますが、前のページの例の続きとして、あるオブジェクトが TARO オブジェクトにぶつかったら TARO オブジェクトを削除するにはソース 1 のようにします。

ソース 1: あるオブジェクトが TARO オブジェクトにぶつかったら TARO オブジェクトを削除する例 
Actor actor = getOneObjectAtOffset( 0, 0, TARO.class );
if( actor != null ){
    getWorld().removeObject( actor ); // getWorld() は World オブジェクトの getter
}
