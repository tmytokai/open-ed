Actor のサブクラスのオブジェクトをプログラム中で動的に作成して画面に配置するためには、そのオブジェクトを new してから World クラスの addObject メソッドを利用します。

Actor のサブクラスのオブジェクトを画面に配置 
public void World.addObject( オブジェクト, x 座標, y 座標 )

戻り値(void): 無し。

例えばあるクラスの任意のメソッドの中で TARO クラスのオブジェクトを new してから座標 (300,200) に配置するには、World オブジェクトの getter である getWorld メソッドを経由して ソース 1 の様にします。

ソース 1: あるクラスのメソッドの中でオブジェクトを動的配置する例 
getWorld().addObject( new TARO(), 300, 200 ); // getWorld() は World オブジェクトの getter

なお World のサブクラスの場合は getWorld を使う必要はないので、ソース 2 の様に直接 addObject を呼び出せます。

ソース 2: World のサブクラスのメソッドの中でオブジェクトを動的配置する例 
addObject( new TARO(), 300, 200 );
