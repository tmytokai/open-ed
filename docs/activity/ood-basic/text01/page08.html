ある Actor のサブクラスと他のサブクラスのオブジェクトが重なっているか判定するには、Actor クラスの getOneObjectAtOffset メソッドを利用します。

ある Actor のサブクラスと他のサブクラスのオブジェクトが重なっているか判定 
protected Actor Actor.getOneObjectAtOffset( 0, 0, クラス名 )

戻り値(Actor): もし指定したクラスのオブジェクトと重なっていたらそのオブジェクトが戻る。重なって無ければ null が戻る。

このメソッドを使うと、いわゆる当たり判定処理が可能になります。

例えばあるオブジェクトの act などのメソッド内で TARO オブジェクトとの当たり判定を行うにはソース 1 のようにします。

ソース 1: あるオブジェクトが TARO オブジェクトにぶつかった時の当たり判定の例 
Actor actor = getOneObjectAtOffset( 0, 0, TARO.class );
if( actor != null ){
    // TARO とぶつかった時の処理を書く
}
