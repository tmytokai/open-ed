演習 1-5 (個人): 演習1-4で作成したクラス図「CDIAG_1_4」を元にコーディングしてみましょう。

フォルダ「CDIAG_1_5」を新規作成し、その中に移動します
FRUIT.java、ORANGE.java、MyWorld.java のコードを作成します。
以下の Main.java を作って下さい。 

public class Main{

	public static void main(String[] args) {
		MyWorld myworld = new MyWorld();
		// ※  (この行は演習1-6で使います)
		myworld.get_orange0().show();
		myworld.get_orange1().show();
		myworld.get_orange2().show();
		myworld.get_orange3().show();
	}
}

コンパイルして実行します。
チーム内で答え合わせをして下さい。 

結果:

みかん No.0美味い
みかん No.1美味い
みかん No.2美味い
みかん No.3美味い
