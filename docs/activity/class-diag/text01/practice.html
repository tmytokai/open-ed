演習 1-6 (個人、20 分): 演習1-5で作成した「CDIAG_1_5」において、 カプセル化を徹底するにはなるべく getter を定義したくないので、getter を取り除いてみましょう。

「CDIAG_1_5」をフォルダごとコピーし、フォルダ名を「CDIAG_1_6」とします。
MyWorld.java を開き、orange0〜3 の getter を削除し、代わりに次の public な act メソッドを MyWorld クラスに追加します。 

public void act()
{
    orange0.show();
    orange1.show();
    orange2.show();
    orange3.show();
}

Main.java を以下に置き換えて下さい。 

public class Main{

	public static void main(String[] args) {
		MyWorld myworld = new MyWorld();
		// ※  (この行は演習1-10で使います)
		myworld.act();
	}
}

コンパイルして実行します。
チーム内で答え合わせをして下さい。
指定場所に「FRUIT、ORANGE、MyWorld、Mainクラスのソースコード」と「実行結果」を記入して下さい。

図1: HUMAN クラス図
