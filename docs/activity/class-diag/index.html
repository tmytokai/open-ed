目的

あなたがクラス図とオブジェクト図を描ける技術者になること

必要性

クラス図とオブジェクト図を描くと、クラスやインスタンス(オブジェクト)間の関係や構造を簡潔に示すことが出来ます。
従ってバグが少なくてエレガントなプログラミングをするために、これらの図の描き方を技術者は学ぶ必要があります。

進め方
