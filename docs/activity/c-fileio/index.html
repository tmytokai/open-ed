オープンエデュケーションに戻る

所属コース: IoT の基礎

上位ナレッジ(ナレッジマップ): 無し

アクティビティ: C 言語によるファイル入出力

時限数: 2
対象年齢: 16〜20歳
教育目的: 教養教育(0%)、技術者育成(100%)
最終更新日: 2016/11/10

目的

あなたが C 言語でファイルの入出力が出来る技術者になること

必要性

C 言語では fopen 関数を使うことでプログラム内でファイルを読み書きすることが可能です。
ファイルを仲介してプログラム外とデータをやりとりするために、C 言語によるファイル入出力の方法を技術者は学ぶ必要があります。

進め方

もし上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
またこのアクティビティの具体的な進め方についてはこちらを参照して下さい。

スケジュール

このアクティビィティのスケジュールは以下の通りです。

時限学習項目内容
1-2 [1] ファイル入出力
  1. ファイルのオープン
  2. ファイルのクローズ
  3. エラー処理
  4. 変数をテキストファイルとして読み書き
  5. 文字列配列をテキストファイルとして読み書き
  6. 配列をバイナリファイルとして読み書き
  7. 構造体をバイナリファイルとして読み書き
