目的

あなたが浮動小数点数について理解出来る技術者になること

必要性

浮動小数点数はコンピュータで実数を扱うためのデータ形式です。
浮動小数点数の特性を知らないとプログラムの動作に問題が生じる場合があるため、浮動小数点数について技術者は学ぶ必要があります。

進め方

上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
アクティビティの具体的な進め方についてはこちらを参照して下さい。
