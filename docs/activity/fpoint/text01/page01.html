1. アクティビティ: 浮動小数点数
  2. 学習項目: [1] 10進数から2進数への基数変換

まず 0 以上の 10 進数の整数を 2 進数に基数変換する方法について説明します。
これは次のようにして行うことができます。

0 以上の 10 進数の整数を 2 進数に変換

A を 0 以上の 10 進数の整数とする

  1. A を 2 で割る。この時の商を $q_0$、余りを $r_0$ とする
  2. $q_0$ を 2 で割る。この時の商を $q_1$、余りを $r_1$ とする
  3. $q_1$ を 2 で割る。この時の商を $q_2$、余りを $r_2$ とする
  4. 商が 0 になるまで以上を繰り返す。最後の商を $q_{n-1}$、余りを $r_{n-1}$ とする
  5. 余りを $r_{n-1}$ を先頭に $r_0$ まで逆順に並べる
  6. 0b を先頭に付ける ※ 2 進数を意味する接頭語
  7. n ビットの 2 進数 0b $r_{n-1}r_{n-2}\cdots r_1r_0$ が得られる

次は逆に 2 進数を 10 進数の整数に基数変換します。

2 進数を 0 以上の 10 進数の整数に変換

n ビットの 2 進数が 0b $r_{n-1}r_{n-2}\cdots r_1r_0$ で与えられる時、

A = $r_{n-1}\cdot2^{n-1} + r_{n-2}\cdot2^{n-2} + \cdots r_1\cdot2^1 + r_0\cdot2^0$

と 10 進数が得られる

では例として 10 進数 123 を 2 進数に変換してみます。

例: 123 を 2 進数に変換

A = 123 なので

  1. 123 を 2 で割る。商は $q_0$ = 61、余りは $r_0$ = 1
  2. $q_0$ = 61 を 2 で割る。商は $q_1$ = 30、余りは $r_1$ = 1
  3. $q_1$ = 30 を 2 で割る。商は $q_2$ = 15、余りは $r_2$ = 0
  4. $q_2$ = 15 を 2 で割る。商は $q_3$ = 7、余りは $r_3$ = 1
  5. $q_3$ = 7 を 2 で割る。商は $q_4$ = 3、余りは $r_4$ = 1
  6. $q_4$ = 3 を 2 で割る。商は $q_5$ = 1、余りは $r_5$ = 1
  7. $q_5$ = 1 を 2 で割る。商は $q_6$ = 0、余りは $r_6$ = 1
  8. 商が 0 になったので計算終了
  9. 余りを $r_{n-1}$ を先頭に $r_0$ まで逆順に並べる
  10. 0b を先頭に付ける
  11. 7ビットの 2 進数 0b1111011 が得られる

逆に 0b1111011 を 10 進数の 0 以上の整数に変換してみます。

例: 0b1111011 を 10 進数の 0 以上の整数に変換

A = $1\cdot 2^{6} + 1\cdot 2^{5} + 1\cdot 2^{4} + 1\cdot 2^{3} + 0\cdot 2^{2} + 1\cdot 2^{1} + 1\cdot 2^{0}$

　= 64 + 32 + 16 + 8 + 0 + 2 + 1 = 123
