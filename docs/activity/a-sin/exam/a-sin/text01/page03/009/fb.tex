\noindent \textbf{【出題意図】}

\noindent 交流電圧の実効値の知識を確かめる問題である。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【重要事項】}

\medskip
振幅 = 実効値$\times \sqrt{2}$

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【解説】}

\noindent 実効値 = 100$\times \sqrt{2} / \sqrt{2} = 100$ [V] が答えとなる。
