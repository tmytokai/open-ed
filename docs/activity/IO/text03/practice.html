ANSELx ・・・ 使用するピンを1(アナログ)にする
TRISx ・・使用するピンを1(出力)にする

ADCON0 = 0b 0 xxxxx 0 1 ・・・ ADC CONTROL REGISTER 0, p119
ADCON1 = 0b 1 000 00 00 ・・・ ADC CONTROL REGISTER 1, p120
ADCON2 = 0b 0000 0000 ・・・ ADC CONTROL REGISTER 2, p121

アクィジョンタイム（コンデンサの充電待ち時間）は __delay_us(20) くらい

ADCON0bits.GO_nDONE = 1; で ADC 開始
while (ADCON0bits.GO_nDONE) continue; で終了待ち

ADRESH ・・・ ADC RESULT REGISTER HIGH, p122
ADRESL ・・・ ADC RESULT REGISTER LOW, p122
