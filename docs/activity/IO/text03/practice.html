2. 出力


演習 3-1 (個人PC、20分): Arduino UNO の アナログ出力ピンから出力される電圧を測定してみましょう。

仕様は以下の通りです。

仕様:

(1) 任意のアナログ出力ピン(※) → 適当な抵抗 → GND とつなぐ。

(2) 2000ミリ秒毎に 0V → 2.5V → 5V → 0V → 2.5V → … と出力値を変える。

　※ 実は Arduino UNO には DAC が内蔵されていないためアナログ出力ピンはありません。
　その代わり GPIO の 3、5、6、9、10、11 ピンから PWM を使って擬似アナログ出力を得られます。
　PWM の周波数は5、6ピンが 980 Hz、その他は 490 Hzです。


仕様に従って回路を組んで下さい。
回路を組んだらファシリテータの確認を受けて下さい。
プログラムを書いて Arduino UNO に転送して下さい。
抵抗の両端にかかっている電圧を測定して正しく動作しているか確認して下さい。
動作したらファシリテータの確認を受けて下さい。
ソースコードをドキュメントにコピーして下さい(回路の写真は必要ありません)。


3. 入力


演習 3-2 (個人PC、20分): Arduino UNO のアナログ入力ピンに温度センサをつないで温度を測定してみましょう。

仕様は以下の通りです。

仕様:

(1) 任意のアナログ入力ピンに温度センサ(lm61ciz)を接続する。

(2) Arduino UNO に入力されている電圧(をmap関数を使って予測した値※)、及び現在の室温(をmap関数を使って予測した値)をシリアルモニタ(虫めがねボタン)に 100 ミリ秒おきに出力する。

※ VsとGNDの間が正確に5Vとは限らないのでVsとGNDの間をテスターで測ってから map 関数で理論的な入力電圧値を求めて下さい。
また理論的な入力電圧値が実際の入力電圧値に近いことも確認して下さい。

データシートや Web 等で回路図を調べて回路を組んで下さい。
回路を組んだらファシリテータの確認を受けて下さい。
プログラムを書いて Arduino UNO に転送して下さい。
シリアルモニタに表示される温度が現在の室温と大体合っていることを確認して下さい。
指で温度センサをつまんでシリアルモニタに表示される温度が変化することを確認して下さい。
動作したらファシリテータの確認を受けて下さい。
ソースコードをドキュメントにコピーして下さい。
回路の全体写真を撮ってドキュメントファイルに貼りつけて下さい。
ドキュメントファイルに「回路の全体写真」と「ソースコード」が記入されていることを確認して下さい。
