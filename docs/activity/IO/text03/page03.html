3. 入力

逆にマイコンへの入力をアナログ電圧としたい場合を考えます。

アナログ電圧をマイコンの内部で扱っているディジタル信号(整数のディジタル値)に変換するためには、出力時の PWM の様な便利な方法は無いので必ず ADC が使われます。
なお電圧が入力されるピンの事を「アナログ入力ピン」と呼びます。

さてアナログ入力ピンに入力されたアナログ電圧をディジタル値に変換する関数の書式は大抵は次のようになっています。

アナログ電圧をディジタル値に変換して取得する関数の書式:

関数名( アナログ入力ピンの番号 )

戻り値: 出力ディジタル値

入力電圧値や出力ディジタル値の範囲はやはり関数仕様によってまちまちになります。
