  2. 学習項目: [2] UART/RS-232C

ある機器から別の機器に UART/RS-232C を使ってデータを出力したい場合は、送信側機器の Tx ピンと受信側機器の Rx ピンを線 1 本で繋ぐだけで可能です。

ただし UART と RS-232C は入出力の電圧レベルが異なるため、回路内に UART と RS-232C が混在している場合は電圧レベルに注意する必要があります。
具体的には、UARTは 0V 〜 3.3V/5V ですが、RS-232C の電圧レベルは -15V 〜 +15V になります。

さらに細かい事をいうと信号論理も逆になっていて、UART は 0V がfalse(0)、3.3V/5V がtrue(1)ですが、RS-232Cは＋がfalse(0)、−がtrue(1)になっています。

従って UART と RS-232C の間に電圧レベル変換回路を入れないと通信出来ないばかりか、機器や回路が壊れる事もあるので注意して下さい。
