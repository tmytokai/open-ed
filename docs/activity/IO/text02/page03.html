  2. 学習項目: [2] RS-232C

入力の場合も送信側の Tx ピンと受信側の Rx ピンを線1本で繋ぎ、メーカーが用意したライブラリの関数を使って情報を読み込むだけです。

関数の書式は大抵は次のようになっています。

シリアル通信で情報を受け取る:

関数名( ポート番号(※) )

戻り値: データ( char型変数 1 byte 分、文字列バッファのアドレスなど )

※マイコンボードやICの場合はシリアルポートがひとつしか無いことが多いのでポート番号は省略されることもある

出力の時と同様に電圧レベルには十分注意して下さい。
