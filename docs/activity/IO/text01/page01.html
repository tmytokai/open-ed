1. GPIOとは

GPIO は General Purpose Input Output(汎用入出力) の略で、HIGHT/LOW とか ON/OFF とか 1/0 とか TRUE/FALSE とかのディジタル入出力を行うインターフェースで、一般的にはポート形式かピン形式で提供されています(※)。

※ これ以降はピン形式で提供されていると仮定して GPIO ピンと書きます。

なお GPIO ではなく「ディジタル入出力」と呼ばれることもあります。

PCなどに GPIO のピンが付いている事はあまりありませんが、大抵のマイコンボードやPICなどには GPIO ピンが付いていて、ライブラリを通じてソフトウェア上から GPIO ピンを簡単に制御することが出来ます。
