  2. 学習項目: [1] GPIO

演習時間: 50 分


今回の演習環境は以下の通りです。

演習環境:

  • PIC : PIC16F1827-I/P (18ピン)
  • PICの設定: 3.3V 駆動、FOSC = 4 Mhz
  • IDE: MPLAB X IDE
  • コンパイラ: XC8
  • プログラマ: MPLAB PICkit4

1. GPIOとは

演習 1-1 (個人PC、30分): PIC にプログラムの書き込みをしてみましょう。

IDE を起動して新規プロジェクトを作成します。デバイスは「Mid-Range 8-bit MCUs → PIC16F1827」、コンパイラは「XC8」、プロジェクト名は「IO_1_1」、エンコーディングは「UTF-8」、それ以外はデフォルト設定のままにしてください。
Soucre Files フォルダにソースファイルを追加します。ファイル名は 「IO_1_1」としてください(拡張子 .c は自動で付きます)。
IO_1_1.c を開き、ファイルの中身をテンプレートの内容に置き換えてください。

この演習で気をつける必要がある設定は以下の2つです。
これ以外の設定についてはコード内のコメント、もしくはデータシートを参考にしてください 

設定 1:

OSCCON  ※ 8bit レジスタ

(解説)
オシレータ(発振回路)の設定で、FOSC(Frequency of Oscillator)を 4 Mhz にセットしています
今回は FOSC = 4 MHz (SPLLEN=0, IRCF=0b1101) としています
詳しくはデータシート 67p を参照してください


設定 2:

_XTAL_FREQ  ※ マクロ

(解説)
__delay_ms 関数を使う場合は define する必要があります
OSCCONの設定と合わせてください(今回は 4 Mhz)

コンパイルボタンを押してエラーが無いことを確認して下さい。
今回は PICkit から電源を供給します。プロジェクトのプロパティを開き、「PICkit4 → Power」選び、「Power target 〜」をチェックし、「Voltage Level」を 3.3 にしてください。
回路を作成します。

この時点では PIC のピン配置設定と回路接続方法は以下の通りなっていますので、以下の設定に従って回路を作成して下さい。
ピン配置について詳しくはデータシートの 5p 、PICKit の詳細については PICKit のマニュアルを参照して下さい。 

ピン配置設定・回路接続方法:

1: RA2
2: RA3
3: RA4
4: MCLR/VPP → PICkit の 1: NMCLR ※ PIC内部で弱プルアップしてるのでプルアップ抵抗必要無
5: VSS → GND ← PICkit の 3: GND
6: RB0
7: RB1
8: RB2
9: RB3

10: RB4
11: RB5
12: ICSPCLK → PICkit の 5: PGC
13: ICSPDAT → PICkit の 4: PGD
14: VDD → 3.3V ← PICkit の 2: VDD
15: RA6
16: RA7
17: RA0
18: RA1

書き込みボタンを押して正常に書き込みできる事を確認して下さい。
指定場所に「正常書き込み完了時のメッセージ」をコピーして記入して下さい。


2. 出力

演習 1-2 (個人PC、30分): GPIO を使って LED を光らせてみましょう。プログラムの仕様は以下の通りです。

仕様:

  1. RA0 (17ピン) に LED を接続する
  2. LEDが 100 ミリ秒光る → 100 ミリ秒消える → 100 ミリ秒光る ・・・を繰り返す

プロジェクト「IO_1_1」を「IO_1_2」にコピーします。
プロジェクト IO_1_1 を閉じます。
「IO_1_1.c」を「IO_1_2.c」に名前を変えます。
(もし開いていたら IO_1_1.c を閉じて) IO_1_2.c を開きます。
プログラムを作成します。

まず古いmain関数を消します。
それからテンプレートの内容をファイルの末尾にコピーしてください (穴埋め問題になっていますので？のところを正しい内容に修正してください)

この演習で気をつける必要がある設定は以下の2つです。
今回はポートA、B全てがデジタル出力モードにセットされています。 

設定 1:

TRISA  ※ 8bit レジスタ
TRISB  ※ 8bit レジスタ

(解説)

ポート A,B の入出力設定です
0 にすると入力モード、1 にすると出力モードになります
詳しくはデータシート 120p(ポートA)と126p(ポートB) を参照してください


設定 2:

ANSELA  ※ 8bit レジスタ
ANSELB  ※ 8bit レジスタ

(解説)

ポート A,b のアナログ設定です
0 にするとデジタル入出力モード、1 にするとアナログ入力モードになります
詳しくはデータシート 122p(ポートA)と128p(ポートB) を参照してください

回路を作成します。

以下の設定に従って回路を作成して下さい。
(*) が前の演習からの変更箇所です。

2: RA3
3: RA4
4: MCLR/VPP → PICkit の 1: NMCLR ※ PIC内部で弱プルアップしてるのでプルアップ抵抗必要無
5: VSS → GND ← PICkit の 3: GND
6: RB0
7: RB1
8: RB2

15: RA6
16: RA7
(*)17: RA0 (出力) → LED → 抵抗 → GND
18: RA1

書き込みボタンを押して正常に動作する事を確認して下さい。

3. 入力

演習 1-3 (個人PC、30分): GPIO を使ってデジタル入力をしてみましょう。プログラムの仕様は以下の通りです。

仕様:

  1. RB0 (6ピン) にタクトスイッチを接続する
  2. タクトスイッチを押してない時は LED が光ってる
  3. タクトスイッチを押すと LED が消える

プロジェクト「IO_1_2」を「IO_1_3」にコピーします。
プロジェクト IO_1_2 を閉じます。
「IO_1_2.c」を「IO_1_3.c」に名前を変えます。
(もし開いていたら IO_1_2.c を閉じて) IO_1_3.c を開きます。
プログラムを作成します。

まず古いmain関数を消します。
それからテンプレートの内容をファイルの末尾にコピーしてください (穴埋め問題になっていますので？のところを正しい内容に修正してください)

この演習で気をつける必要がある設定は以下の2つです。
今回は RB0 をデジタル入力モードにセットします。 

設定 1:

WPUB  ※ 8bit レジスタ

(解説)

ポートBの(弱)プルアップ抵抗の設定です
詳しくはデータシート 126p を参照してください


設定 2:

OPTION_REGbits.nWPUEN  ※ 8bit レジスタ OPTION_REG の bit 7

(解説)

PICの内部(弱)プルアップ抵抗機能を ON にする設定です
ローアクティブなので 0 をセットすると ON になりす
内部プルアップ抵抗機能を使いたい場合は忘れず ON にしてください
詳しくはデータシート 175p を参照してください
