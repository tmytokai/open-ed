オープンエデュケーションに戻る

所属コース: IoT の基礎

上位ナレッジ(ナレッジマップ): 無し

アクティビティ: 入出力インターフェース

時限数: 16
対象年齢: 16〜20歳
教育目的: 教養教育(0%)、技術者育成(100%)
最終更新日: 2016/11/6

目的

あなたが入出力インターフェースを使ってデバイスを制御したりデバイス間で情報をやり取りできる技術者になること

必要性

GPIOやRS-232Cなどの入出力インターフェースはコンピュータや周辺機器などのデバイスを制御したり、デバイス間で情報をやり取りするための技術です。
うまくデバイスを活用するため、各種入出力インターフェースの使い方を技術者は学ぶ必要があります。

進め方

もし上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
またこのアクティビティの具体的な進め方についてはこちらを参照して下さい。
