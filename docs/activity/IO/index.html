目的

あなたが入出力インターフェースを使ってデバイスを制御したりデバイス間で情報をやり取りできる技術者になること

必要性

GPIO、UART/RS-232C、SPI、I2C などの入出力インターフェースはコンピュータや周辺機器などのデバイスを制御したり、デバイス間で情報をやり取りするための技術です。
うまくデバイスを活用するため、各種入出力インターフェースの使い方を技術者は学ぶ必要があります。

進め方

上位ナレッジがある場合は、このアクティビティを始める前にそちらから学んで下さい。
アクティビティの具体的な進め方についてはこちらを参照して下さい。
