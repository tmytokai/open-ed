  2. 学習項目: 学習項目: [5] 割り込み処理

割り込み処理とは、コンピュータがプログラムを実行している最中に何かしらのイベントが生じた時に、プログラムの実行を一時中断してイベントに対応する処理を行う機能のことです。

割り込みイベントの種類としてはコンピュータの外部機器や内部モジュールからの割り込みである「外部割り込み(またはハードウェア割り込み)」と、実行中のプログラム内からの割り込みを処理する「内部割り込み(またはソフトウェア割り込み)」があります。

外部割り込みのイベントとしては「INT 割り込み」や「タイマー割り込み」などがあります。
一方、内部割り込みのイベントとしては「ゼロ除算例外」や「保護されたメモリへの不正アクセス例外」などがあります。

ただし一般的なマイコンでは内部割り込みはサポートしていない(みたい)なので(※)、マイコンで割り込みと言えば大体は外部割り込みの事を意味します。

※ どうしてもプログラム内から無理矢理内部割り込みをかけたい場合は、下のコード様に何か適当な割り込み処理を有効にしてから割り込みフラグを手動でONにします 

INTCONbits.INTF = 0;
INTCONbits.INTE = 1 ; // INT 割り込み有効化
INTCONbits.GIE = 1;

〜

INTCONbits.INTF = 1; // 割り込みイベントを手動発生
