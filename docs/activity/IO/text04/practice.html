演習


1. Bluetooth とは

演習 1 (個人PC、10分): Bluetooth プロファイルについて調べてみましょう。

「自分が常日頃からお世話になっている Bluetooth 機器とそのプロファイルの組」を最低ひとつ Web 等で調べて下さい。
「調べた結果」をドキュメントファイルに記述して保存してください。


2.出力


演習 2 (個人PC、40分): Arduino UNO から Bluetooth 経由でタブレットに文字列を送ってみましょう。

仕様は以下の通りです。

仕様:

(1) Bluetooth 機器として「ＲＮ－４２使用Ｂｌｕｅｔｏｏｔｈ無線モジュール評価キット」を使用する。

(2) プロファイルは SPP を使用する。

(3) タブレット(Nexus 9)側の通信アプリとして Bluetooth spp pro を使う。

(4) 100ミリ秒毎に Arduino UNO 側からタブレット側へ適当な文字列が送られる。

ホワイトボードに回路図を描いて下さい。
回路図を描いたらファシリテータの確認を受けて下さい。
回路図の写真を撮ってドキュメントファイルに貼りつけて下さい。

　(注意) 次の課題でも殆ど同じ回路図を描きますのでホワイトボードは消さないで下さい。

設計図を見ながら回路を組んで下さい。
回路を組んだらファシリテータの確認を受けて下さい。
回路の全体写真を撮ってドキュメントファイルに貼りつけて下さい。
プログラムを書いて Arduino UNO に転送して下さい。

　(注意) Arduino UNO 側のRX/TXピンのジャンパケーブルを外さないとエラーが出ます。

タブレットとの通信を確認して下さい。
動作したらファシリテータの確認を受けて下さい。
ソースコードをドキュメントにコピーして下さい。
ドキュメントファイルに「回路図の写真」と「回路の全体写真」と「ソースコード」が記入されていることを確認して下さい。


3.入力


演習 3 (個人PC、20分): Arduino UNO を Bluetooth 経由でタブレットから制御してみましょう。

仕様は以下の通りです。

仕様:

(1) 回路は演習2とほぼ同じだが Arduino UNO の適当な GPIO に LED をつなげる。

(2) プロファイルは SPP を使用する。

(3) タブレットから 'a' が送られたら LED を点灯する。

(4) タブレットから 'b' が送られたら LED を消灯する。

ホワイトボードに回路図を描いて下さい。
回路図を描いたらファシリテータの確認を受けて下さい。
回路図の写真を撮ってドキュメントファイルに貼りつけて下さい。
設計図を見ながら回路を組んで下さい。
回路を組んだらファシリテータの確認を受けて下さい。
回路の全体写真を撮ってドキュメントファイルに貼りつけて下さい。
プログラムを書いて Arduino UNO に転送して下さい。

　(注意) Arduino UNO 側のRX/TXピンのジャンパケーブルを外さないとエラーが出ます。

タブレットとの通信を確認して下さい。
動作したらファシリテータの確認を受けて下さい。
ソースコードをドキュメントにコピーして下さい。
ドキュメントファイルに「回路図の写真」と「回路の全体写真」と「ソースコード」が記入されていることを確認して下さい。
