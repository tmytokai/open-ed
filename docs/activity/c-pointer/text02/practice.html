1. 変数とメモリ空間

演習 1 (個人PC、20分): sizeof 演算子を使って char型、short型、int型、double 型の変数 1 個がメモリ空間内で使用するデータサイズ(byte)を調べてみましょう。
この演習では sizeof(型の名前) の記述を使ってデータサイズを printf で 10進数表示して下さい。
ドキュメントファイルに「ソースコード」と「実行結果」をコピーし、「各型のデータサイズ(byte)」を記述してから保存してください。

演習 2 (個人PC、10分): sizeof 演算子を使って char型の変数 a、short型の変数 b、int型の変数 c、double 型の変数 d がメモリ空間内で使用するデータサイズ(byte)を調べてみましょう。
この演習では各変数を順に定義してから sizeof(変数の名前) の記述を使ってデータサイズを printf で 10進数表示して下さい。
ドキュメントファイルに「ソースコード」と「実行結果」をコピーし、「各変数のデータサイズ(byte)」を記述してから保存してください。

演習 3 (個人PC、10分): アドレス演算子を使って char型の変数 a、short型の変数 b、int型の変数 c、double 型の変数 d のアドレスを調べてみましょう。
演習2のソースをコピーし、データサイズを printf で 10進数、各変数のアドレスを16進数で表示して下さい。
なお printf でアドレスを 16進数表示する時は %p を使います。
ドキュメントファイルに「ソースコード」と「実行結果」をコピーし、「各変数のアドレス」を記述してから保存してください。

演習 4 (個人PC、20分): 演習 3 の各変数がメモリ空間内でどの様に配置されているか視覚的に確認してみましょう。
以前のテキストを参考にして、表計算ソフトを使ってメモリ空間を書いて下さい。
もしアドレスの桁数が大きすぎる場合は下4桁だけで良いです。
ドキュメントファイルに「メモリ空間」をコピーして保存してください。
