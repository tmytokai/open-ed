\noindent \textbf{【出題意図】}

\noindent bit, byte, KiB, MiB の単位を相互変換できるかどうかを確かめる問題である。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【重要事項】}

\medskip
\noindent ・ 1 byte = 8 bit

\medskip
\noindent ・ 1 KiB = 1024 byte

\medskip
\noindent ・ 1 MiB = 1024 KiB

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\noindent \textbf{【解説】}

\noindent 重要事項に示した関係より答は求まる。
