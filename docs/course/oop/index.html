目的

あなたがオブジェクト指向プログラミングができる技術者になること

必要性

Java, JavaScript, PHP, C#, C++, Objective-C, Swift など、現在主流となっているプログラミング言語はほぼ例外なくオブジェクト指向言語です。
従って技術者はオブジェクト指向プログラミングを必ず学ぶ必要があります。

進め方

このコースの目的はコースを構成する複数のアクティビティを実施することで達成されます。
コースやアクティビティ間の関連についてはナレッジマップを参照して下さい。
なお各アクティビィティの具体的な内容については以下のスケジュール表からそれぞれの詳細ページを参照して下さい。
