上位ナレッジ(ナレッジマップ):

  1. コース: アナログ信号のスペクトル解析

目的

あなたが時間領域ディジタル信号のスペクトル解析を行える技術者になること

必要性

時間毎に変化する自然現象(例えば電流や電圧、気温、気圧、音声、体温、血圧、etc.)はセンサ等を用いて時間領域ディジタル信号に変換出来ます。
従ってこれらの自然現象が起きた背景をコンピュータによって分析して様々な分野で活用するために、技術者は時間領域ディジタル信号のスペクトル解析を学ぶ必要があります。

進め方

このコースの目的はコースを構成する複数のアクティビティを実施することで達成されます。
コースやアクティビティ間の関連についてはナレッジマップを参照して下さい。
なお各アクティビィティの具体的な内容については以下のスケジュール表からそれぞれの詳細ページを参照して下さい。
