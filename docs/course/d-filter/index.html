目的

あなたがディジタル線形フィルタをプログラミングして作成出来る技術者になること

必要性

ローバスフィルタやハイパスフィルタなどのディジタル線形フィルタは様々な分野で利用されている基本技術です。
従って様々な分野で活躍するために、ディジタル線形フィルタの作り方を技術者は学ぶ必要があります。

進め方

このコースの目的はコースを構成する複数のアクティビティを実施することで達成されます。
コースやアクティビティ間の関連についてはナレッジマップを参照して下さい。
なお各アクティビィティの具体的な内容については以下のスケジュール表からそれぞれの詳細ページを参照して下さい。
