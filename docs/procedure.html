アクティビティの進め方

各アクティビティは以下のARCSモデルに従ってデザインされています。

  1. （Attention) 興味のある内容にします。
  2. （Relevance）親しみやすい、または何かに使えそうな内容にします。
  3. （Confidence）やれば出来そうな内容にします。
  4. （Satisfaction）達成感を得られそうな内容にします。

各アクティビティには目的があり、その目的を達成するため以下のフローに従ってあなたはアクティビティを進めます。


フロー

1. 自分のプライペート空間で学習を行なって、あなたは各学習項目の知識を身につけます。

  • クラスルームでの板書は一切ありません。
  • オンラインテキストを用意していますので、いつどこででも自分のペースで好きなだけ自由に学べます。
  • オンラインテキストに用意された学習項目以外の関連項目もネット上や図書館等で自由に好きなだけ学んで下さい。
  • オンラインテキストに関して質問や意見等があればファシリテータにオンライン・オフライン問わずいつでも何でもお尋ね下さい。

2. クラスルーム内で演習を行って、あなたは各学習項目に対する理解を深めます。

  • 3〜4人でチームを組んで演習を行います。
  • 演習は「チーム演習」、「個人演習」の 2 種類に分かれます。
  • 演習中はファシリテータがクラスルーム内を周って快適に演習が出来るようサポートします。
  • オンラインダッシュボードで各チームの進行状況を可視化します。

3. オンライン上で達成度評価試験を行って、あなたは各学習項目に対する自身の理解度を確認します。

  • アクティビティによっては達成度評価試験が無い場合もあります
  • 達成度評価試験修了後、即座に採点と問題解説を行います。

4. リフレクションを行なって、あなたは各学習項目を学んだことを社会に証明し、アクティビティの改善に貢献します。

  • リフレクションで挙げられた改善点をフィードバックしてアクティビティを改善します。。

「チーム演習」の進め方

「チーム演習」は調べ物やディベートなど、チームで行う演習です。

各メンバーはクラウド上で共有されているチームレポートを開いて下さい。
各メンバーは自分の「出席番号」「氏名」を書き入れて下さい。
演習問題の指示に従って解答をチームで考えて下さい。
ファシリテータの指示に従って指定された場所に解答して下さい。
過半数の解答がそろったらファシリテータが解説します。
ファシリテータの解説を聞いて何か新しい発見があったら解答を適宜修正して下さい。
ファシリテータは提出状況を確認してダッシュボードにチェックを入れます。


「個人演習」の進め方

「個人演習」は計算やプログラミングなど、まず個人で演習した後にチームで教え合いを行う演習です。

クラウド上にある個人レポートを開いて下さい。
演習問題の指示に従ってまずは自力で問題を解いて下さい。
解き終わったメンバーはチーム内で終わってないメンバーに解き方を教えて下さい。
分からないメンバーはチーム内の他のメンバーに質問して下さい。
ファシリテータの指示に従って指定された場所に解答して下さい。
過半数の解答がそろったらファシリテータが解説します。
ファシリテータの解説を聞いて何か新しい発見があったら解答を適宜修正して下さい。
ファシリテータは提出状況を確認してダッシュボードにチェックを入れます。


「リフレクション」の実施

全ての演習課題を終えたら「リフレクション」を行います。
フォームの指示に従って入力して下さい。
