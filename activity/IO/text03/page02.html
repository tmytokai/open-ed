出力

ここではもっと具体的に、マイコンの出力をアナログ電圧で得たい場合を考えます。

マイコンの内部で扱っているディジタル信号(整数のディジタル値)をアナログ電圧に変換して出力する方法として前のページで述べた DAC を使う方法と GPIO に PWM 出力させる方法があります(※)。

※ もちろん DAC を使う方が PWM を使う方よりも安定した電圧が出るのですがコスト的には高くなります。

いずれにしろ、電圧が出力されるピンの事を「アナログ出力ピン」と呼びます。

さてアナログ出力ピンからアナログ電圧を出力する関数の書式は大抵は次のようになっています。

ディジタル値をアナログ電圧に変換して出力する関数の書式:

関数名( アナログ出力ピンの番号、入力ディジタル値 )

入力する整数のディジタル値の範囲は 0〜255 とか 0〜1024 のように関数仕様によりまちまちになります。
また出力される電圧値も 0〜3.3V とか 0〜5V のように関数仕様によりまちまちになります。
ただ一般的には線形変換(※)になっていることが多いです。

※ 例えば入力が 0 〜 255 の整数で出力が 0 〜 5V の電圧なら、入力が 0 のとき 0V、入力が 255 の時 5V が出力されること。
