アクティビティ: 入出力インターフェース
学習項目: [3] DAC と ADC
演習時間: 60 分

演習


1. DAC と ADC とは

演習 1 (個人PC、10分): 「自分が常日頃からお世話になっているDACとADCの例」を最低ひとつずつ考えてドキュメントファイルに記述してください。


2.出力

演習 2 (個人PC、20分): Arduino UNO の アナログ出力ピン(PWM出力)から出力される電圧を測定してみましょう。

仕様:

(1) 任意のアナログ出力ピン → 適当な抵抗 → GND とつなぐ

(2) 2000ミリ秒毎に 0V → 2.5V → 5V → 0V → 2.5V → … と出力値を変える

まずホワイトボードに回路の設計図を描いて下さい。
チームメンバー全員が描いたらファシリテーターの確認を受け、自分が描いた設計図の写真を撮って保存して下さい。

次に設計図を見ながら回路を回路を組んで下さい。
チームメンバー全員が組んだらファシリテーターの確認を受け、自分が組んだ回路の全体写真を撮って保存して下さい。

次にプログラムを書いて転送し、抵抗の両端にかかっている電圧を測定して下さい。
チームメンバー全員が動作したら各自ファシリテーターの動作確認を受けて下さい。

最後にドキュメントファイルに「設計図の写真」と「回路の写真」と「ソースコード」をコピーして保存してください。


3.入力

演習 3 (個人PC、30分): Arduino UNO のデジタル入力ピンに温度センサをつないで温度を測定してみましょう。

仕様:

(1) 任意のアナログ入力ピンに温度センサ(lm61ciz)を接続する

(2) 現在の温度をシリアルモニタに出力する(※)

※ VsとGNDの間が正確に5Vとは限らないのでVsとGNDの間をテスターで測ってから map 関数で理論的な入力電圧値を求めて下さい。
また理論的な入力電圧値が実際の入力電圧値に近いことも確認して下さい。

まず仕様に従って回路を組んで下さい。
チームメンバー全員が組んだらファシリテーターの確認を受け、自分が組んだ回路の全体写真を撮って保存して下さい。

次にプログラムを書いて転送して下さい。
チームメンバー全員が動作したら各自ファシリテーターの動作確認を受けて下さい。

最後にドキュメントファイルに「設計図の写真」と「回路の写真」と「ソースコード」をコピーして保存してください。
