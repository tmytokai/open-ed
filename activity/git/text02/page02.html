ローカルリポジトリの新規作成


次は git init コマンドを使ってローカルリポジトリを新規作成してみましょう。
今回は例として「gitlocal」という名前のローカルリポジトリを新規作成してみます。 
 (1) 「gitlocal」というフォルダを作る 


 (2) そのフォルダの中に移動する 


 (3) git init

これでローカルリポジトリ「gitlocal」が出来ました。
また gitlocal フォルダそのものはワーキングツリーになりました。

なおワーキングツリーの中に出来た「.git」というフォルダがローカルリポジトリの本体ですが、.git は隠しフォルダ扱いになっているのでそのままでは見えません。
もし .git があるかどうか確認したい場合はLinuxなら「ls -al」、Windowsなら「dir /AH」を実行してください。

さてこの時点で「ワーキングツリー」「インデックス」「HEAD」は次のような状態になっています。
当然ですが「ワーキングツリー」「インデックス」の中身は空で、「HEAD」は存在しません(※)。

※ 正確にはワーキングツリーには .git フォルダが含まれていますが省略します。
