ローカルリポジトリでの作業


次は各開発者が自分のローカルリポジトリ内で自分の作業を行いますが、次の点を常に念頭に置いて作業すると良いでしょう。


「開発者間のローカルリポジトリは完全に独立しているので、自分のローカルリポジトリ内だけで作業している限り、何をしても他の開発者に影響は無い」

さて今回の演習ではhoge.txtに各自の名前(A〜D)を入力し(最後の改行を忘れずに)、git add してから git commit して下さい(commit のコメントは my name is 〜)。
すると以下の状態2になります。

この時点では、他の人やリモートリポジトリに何の影響も与えていないことに注目して下さい。
