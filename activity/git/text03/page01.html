リモートリポジトリのクローン


ここではリモートリポジトリを使って多人数(今回はAさん、Bさん、Cさん、Dさんの4名)で共同開発をする演習を行います。

まずリモートリポジトリを新規作成する必要があるのですが、手作業だと結構面倒な作業になるので(※)今回は省略し、既に存在するリモートリポジトリをクローン(リモートリポジトリをローカルリポジトリとしてコピーすること)して使うことにします。

※ GitHub などの外部ウェブサービスを利用すればブラウザ上で簡単に作れます。 

 (1) 端末(※)を開いてホームディレクトリに移動する


※ Windows環境の場合は 端末 ＝ コマンドプロンプトと読み替えて下さい


 (2)リモートリポジトリをクローンする


git clone <リモートリポジトリのアドレス>


 (3) ls か dir でローカルリポジトリが出来ているか確認する


 (4) ローカルリポジトリに cd する


さて今回の演習で使用するリモートリポジトリ「gitremote」をクローンすると以下の状態1になります。
リモートリポジトリの内容がA〜Dさんのローカル環境にそのままコビーされていることに注目して下さい。

なおリモートリポジトリのところにある「origin」とはリモートリポジトリの別名です。
