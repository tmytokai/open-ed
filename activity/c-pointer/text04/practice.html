1. ポインタ変数

演習 1 (個人PC、10分): 間接演算子の使い方を学びましょう。
int 型の変数 a、 int 型ポインタ変数 pa を定義し、アドレス演算子を使って変数 a のアドレスを pa に代入して下さい。
次に a に整数 100 を代入して下さい。
最後に間接演算子とポインタ変数 pa を使って printf で a に入っている値を表示して下さい。
ドキュメントファイルに「ソースコード」と「実行結果」をコピーして保存してください。

演習 2 (個人PC、10分): 間接演算子の使い方を学びましょう。
int 型の変数 a、 int 型ポインタ変数 pa を定義し、アドレス演算子を使って変数 a のアドレスを pa に代入して下さい。
次に間接演算子とポインタ変数 pa を使って a に整数 100 を代入して下さい。
最後に printf で a の値を表示して下さい。
ドキュメントファイルに「ソースコード」と「実行結果」をコピーして保存してください。

演習 3 (個人PC、10分): ポインタ変数のデータサイズが参照先変数のデータ型によって変わらないことを確認しましょう。
char 型のポインタ変数 pa、short 型のポインタ変数 pb、int型のポインタ変数 pc、double 型のポインタ変数 pd を定義して下さい。
次に sizeof(ポインタ変数名) の記述を使って各ポインタ変数のデータサイズを printf で 10進数表示し、チーム内で何故全てこのサイズになるのか話し合って下さい。
ドキュメントファイルに「ソースコード」と「実行結果」をコピーし、「各ポインタ変数のデータサイズ(byte)」と「ポインタ変数のデータサイズが同じ理由」を記述してから保存してください。

演習 4 (個人PC、10分): 参照先の変数とポインタ変数がメモリ空間内でどの様に実際に配置されているか視覚的に確認してみましょう。
まず演習 2 と同様に int 型の変数 a、 int 型ポインタ変数 pa を定義し、アドレス演算子を使って変数 a のアドレスを pa に代入して下さい。ただし a を 100 で初期化して下さい。
次に a の内容(10進数)とデータサイズ(10進数)とアドレス(16進数)、pa の内容(16進数)とデータサイズ(10進数)とアドレス(16進数) を printf で表示して下さい。
またテキストの図1を参考に、表計算ソフトを使ってメモリ空間を書いて下さい。
もしアドレスの桁数が大きすぎる場合は下4桁だけで良いです。
ドキュメントファイルに「メモリ空間」をコピーして保存してください。
